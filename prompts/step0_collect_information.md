# Step 0: 広域情報収集（7カテゴリ）★情報起点の第一歩

## ミッション

**企業ありきではなく、情報から企業を発見する。**

7カテゴリの広域情報を収集し、全ての情報に**4次元評価**（確度・検証度・非対称性・鮮度）を付与する。

---

## 基本原則

```
情報起点のアプローチ（このシステムの核心）:
★Step 0: 幅広い情報収集 ← 今ここ
    ↓
Step 0.5: 情報から企業発見 → 候補プール50社作成
    ↓
Step 1以降: 企業深掘り → 評価

❌ 企業起点（避けるべき）:
企業リスト → その企業の情報収集 → 評価
```

**重要**: このStep 0が分析の出発点。ここで収集した情報を基に、Step 0.5で企業を発掘する。情報が先、企業が後。

## 4次元評価の付与

全ての収集情報に以下の評価を付与する：

### 確度（Certainty）

| stage | score | 定義 |
|-------|-------|------|
| rumor | 10-20 | 噂、観測記事 |
| report_unconfirmed | 30-40 | 報道あり、一次資料未確認 |
| report_confirmed | 50-60 | 複数報道、一次資料なし |
| official_announcement | 70-80 | 公式発表あり |
| contract_signed | 85-95 | 契約・予算成立 |
| executed | 95-99 | 実行済み |

### 検証度（Verification）

| level | 定義 |
|-------|------|
| primary_confirmed | 一次資料で確認 |
| multi_source | 2ソース以上 |
| single_reliable | 単一の信頼ソース |
| single_unverified | 未検証 |

### 非対称性（Asymmetry）

| score | 状態 |
|-------|------|
| 21-25 | 3源泉全てで非対称性 |
| 16-20 | 2源泉で非対称性 |
| 11-15 | 1源泉で非対称性 |
| 6-10 | 一部認知済み |
| 0-5 | 広く認知済み |

### 鮮度（Freshness）

| decay_rate | 有効期間 |
|------------|----------|
| fast | 当日〜1週間 |
| medium | 1週間〜3ヶ月 |
| slow | 3ヶ月〜1年 |

## 新規企業候補の発見

情報収集中に新たな企業候補を発見した場合：

1. **発見時の記録**:
   ```json
   "new_company_candidates": [
     {
       "ticker": "9999",
       "name": "新発見企業",
       "discovery_source": "intel_E_003",
       "discovery_reason": "大型投資の主要サプライヤーとして言及",
       "recommended_action": "Step 0で追加発掘"
     }
   ]
   ```

2. **候補プールへの追加**:
   - 重要な発見の場合、Step 0の候補プールに追加を推奨
   - next_actionsに記録
