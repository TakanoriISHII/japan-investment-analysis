# Step 6: 最終レポート生成

## ミッション

全データを統合し、投資委員会向けの最終レポートを生成する。
**情報の確度と変化点を明確に示す**ことが最重要。

---

## 入力データ

1. `data/events.json` — イベントログ
2. `data/information_quality.json` — 情報品質サマリー
3. `data/asymmetry_tracker.json` — 非対称性トラッカー
4. `data/snapshots/monopoly_map.json` — 独占マップ
5. `data/snapshots/market_state.json` — 市場状態
6. `data/analysis/market_risk.json` — 市場危険度
7. `data/analysis/top30.json` — Top30スコアリング

---

### 4. 4象限マトリックス

```markdown
【最優先投資】確度高(≥80) × 非対称性高(≥16)
- 8035 東京エレクトロン ⭐最高確信 — 確度92, 非対称性22
- 2802 味の素 — 確度85, 非対称性27（非対称性最高）
- 6857 アドバンテスト — 確度88, 非対称性18
→ コアポジション構築。情報の質が高く、非対称性も残存。

【安定成長】確度高(≥80) × 非対称性低(<16)
- 6146 ディスコ — 確度90, 非対称性12
→ 分散目的で保有。αは小さいが確実性高い。

【投機的機会】確度低(<80) × 非対称性高(≥16)
- 6890 フェローテック — 確度65, 非対称性20
→ 追加調査で確度を上げれば最優先に昇格可能。
→ 要調査: 中国売上比率の一次資料確認

【回避/様子見】確度低(<80) × 非対称性低(<16)
→ 情報不足で判断保留。
```
