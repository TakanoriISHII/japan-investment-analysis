# Step 0: グローバル・ボトルネック機会の動的発掘

## ミッション

**外国人投資家が見落としている「グローバル・ボトルネック支配」企業を大量発掘し、50社の候補プールを作成する。**

### 基本原則

**「30社を選ぶ」のではなく「70-100社から50社を厳選し、Step 5で詳細評価してTop30を決定する」**

見落としリスクを最小化するため、以下のファネルアプローチを採用：

```
Step 0（本ステップ）:
  Phase A: 大量発掘（目標70-100社）
      ↓ 一次スクリーニング
  Phase B: 候補プール作成（50社）
      ↓ 簡易優先度評価
  Phase C: 50社候補プールを独占マップに反映

Step 5（詳細評価）:
  50社候補プール
      ↓ 4軸評価（100点満点）
  Top30選出 + 監視リスト(31-50位)
```

**重要**: Step 0は「候補プール50社」を作成する段階。Top30の最終決定はStep 5で詳細な4軸評価を行った後に行う。

## Phase B: 一次スクリーニング（100社→50社）

### B-1. スクリーニング基準

| 基準 | 条件 | 除外理由 |
|------|------|----------|
| シェア/独占性 | 世界シェア20%以上、または特定領域で独占的 | シェア低すぎ |
| 上場区分 | 東証プライム/スタンダード | 流動性不足 |
| 時価総額 | 100億円以上 | 投資対象外 |
| 情報取得可能性 | 基本的なシェア情報が確認可能 | 情報なし |

### B-2. 簡易4次元チェック

全候補に対して簡易評価を実施：

```
確度チェック:
□ シェア情報に出典があるか
□ 複数ソースで確認可能か

非対称性チェック:
□ 英語メディアで報道されているか（されていなければ高評価）
□ Tier2/3のサプライチェーン深層か
□ 認証障壁があるか
```

### B-3. スクリーニング結果

50社を「候補プール」として選出。ドメインバランスを考慮：

| ドメイン | 候補プール目標 |
|---------|---------------|
| AI | 15-18社 |
| エネルギー | 8-10社 |
| 軍事・防衛 | 7-9社 |
| ロボティクス | 6-8社 |
| 宇宙 | 5-6社 |
| サイバー | 5-6社 |

## Phase C: 簡易優先度評価と候補プール確定（50社）

**注意**: Phase Cは「候補プール50社」を確定する段階。詳細な4軸評価（100点満点）とTop30選出はStep 5で行う。

### C-1. 情報の4次元評価（簡易版）

候補プール50社に対して、情報の質を評価：

#### 確度（Certainty）スコア: 0-100

```
| stage | score範囲 | 定義 |
|-------|-----------|------|
| rumor | 10-20 | 噂、観測記事 |
| report_unconfirmed | 30-40 | 報道あり、一次資料未確認 |
| report_confirmed | 50-60 | 複数報道、一次資料なし |
| official_announcement | 70-80 | 公式発表あり |
| contract_signed | 85-95 | 契約・予算成立 |
| executed | 95-99 | 実行済み、決算で確定 |
```

#### 検証度（Verification）

```
| level | 定義 |
|-------|------|
| primary_confirmed | 一次資料で確認 |
| multi_source | 2ソース以上で確認 |
| single_reliable | 単一だが信頼できる |
| single_unverified | 未検証（減点） |
```

#### 非対称性（Asymmetry）スコア: 0-25

```
基本スコア: 5点
+ 言語障壁（英語報道なし）: +8点
+ サプライチェーン深度（Tier2/3）: +7点
+ 認証障壁（防衛・原子力等）: +5点
```

#### 鮮度（Freshness）

```
| decay_rate | 情報タイプ |
|------------|-----------|
| fast | 株価・需給 |
| medium | 決算・ガイダンス |
| slow | シェア・参入障壁 |
```

### C-2. 簡易優先度の判定

**Step 5での詳細評価に向けた優先度分類**（最終ランキングはStep 5で決定）：

| 優先度 | 条件 | 備考 |
|--------|------|------|
| **high** | 確度≥70 AND 非対称性≥16 | Step 5で「最優先」象限の有力候補 |
| **medium** | 確度≥50 OR 非対称性≥12 | 追加調査でhighに昇格可能 |
| **low** | 上記以外 | ドメインバランスで必要な場合のみ |

### C-3. 候補プール50社の確定

```
1. 50社を選出（Phase Bのスクリーニング通過者）
2. 各社に4次元評価と簡易優先度を付与
3. ドメインバランスを確認
   - 6ドメイン全てで最低3社以上
   - 不足ドメインは追加発掘を推奨
4. 候補プール50社をStep 2で独占マップに反映
```

### C-4. Step 5との連携

**Step 5で行う詳細評価（4軸100点満点）:**

| 評価軸 | 配点 | Step 0での準備 |
|--------|------|----------------|
| グローバル独占力 | 30点 | シェア情報を収集済み |
| 資本効率・財務健全性 | 20点 | **Step 5で財務データを収集** |
| 領域横断シナジー | 25点 | ドメイン情報を付与済み |
| 情報の非対称性 | 25点 | 非対称性スコアを付与済み |

→ Step 0では財務データ（ROE、負債比率等）は収集しない。Step 5で収集して詳細評価を行う。

## 重要な注意事項

### やってはいけないこと

1. **30社ありきで発掘しない** — 必ず70社以上を発掘してから絞り込む
2. **AI・半導体に偏重しない** — 6ドメインから均等に発掘
3. **Tier1だけを見ない** — Tier2/3こそ非対称性が高い
4. **英語情報だけで判断しない** — 日本語情報こそ価値がある
5. **一度の検索で満足しない** — 複数の検索手法を網羅的に実行

### 成功の指標

- 一次発掘で70社以上
- スクリーニング後に50社の候補プール確定
- 候補プール50社の非対称性スコア16以上が15社以上
- 6ドメイン全てで最低3社以上
- Tier2/3企業を候補プールの35%以上含む
- high優先度の企業が20社以上（Step 5での「最優先」象限有力候補）

### Step 5へのインプット品質

| 項目 | Step 0で収集 | Step 5で追加収集 |

| シェア情報 | ✅ | - |

**重要**: Step 0で「候補プール50社」の質を高めることが、Step 5での「Top30選出」の精度を決定する。
