# Step 0: グローバル・ボトルネック機会の動的発掘

## ミッション

**外国人投資家が見落としている「グローバル・ボトルネック支配」企業を大量発掘し、ランキングでTop30を選出する。**

### 基本原則

**「30社を選ぶ」のではなく「70-100社から30社を厳選する」**

見落としリスクを最小化するため、以下のファネルアプローチを採用：

```
Phase A: 大量発掘（目標70-100社）
    ↓ 一次スクリーニング
Phase B: 候補プール（50社）
    ↓ 4次元評価でランキング
Phase C: Top30選出 + 監視リスト(31-50位)
```

## Phase B: 一次スクリーニング（100社→50社）

### B-1. スクリーニング基準

| 基準 | 条件 | 除外理由 |
|------|------|----------|
| シェア/独占性 | 世界シェア20%以上、または特定領域で独占的 | シェア低すぎ |
| 上場区分 | 東証プライム/スタンダード | 流動性不足 |
| 時価総額 | 100億円以上 | 投資対象外 |
| 情報取得可能性 | 基本的なシェア情報が確認可能 | 情報なし |

### B-2. 簡易4次元チェック

全候補に対して簡易評価を実施：

```
確度チェック:
□ シェア情報に出典があるか
□ 複数ソースで確認可能か

非対称性チェック:
□ 英語メディアで報道されているか（されていなければ高評価）
□ Tier2/3のサプライチェーン深層か
□ 認証障壁があるか
```

### B-3. スクリーニング結果

50社を「候補プール」として選出。ドメインバランスを考慮：

| ドメイン | 候補プール目標 |

| AI | 15-18社 |

| 軍事・防衛 | 7-9社 |
| ロボティクス | 6-8社 |

## Phase C: 4次元評価とランキング（50社→Top30＋監視リスト）

### C-1. 詳細4次元評価

候補プール50社に対して、詳細な4次元評価を実施：

#### 確度（Certainty）スコア: 0-100

```
| stage | score範囲 | 定義 |
|-------|-----------|------|
| rumor | 10-20 | 噂、観測記事 |
| report_unconfirmed | 30-40 | 報道あり、一次資料未確認 |
| report_confirmed | 50-60 | 複数報道、一次資料なし |
| official_announcement | 70-80 | 公式発表あり |
| contract_signed | 85-95 | 契約・予算成立 |
| executed | 95-99 | 実行済み、決算で確定 |
```

#### 検証度（Verification）

```
| level | 定義 |
|-------|------|
| primary_confirmed | 一次資料で確認 |
| multi_source | 2ソース以上で確認 |
| single_reliable | 単一だが信頼できる |
| single_unverified | 未検証（減点） |
```

#### 非対称性（Asymmetry）スコア: 0-25

```
基本スコア: 5点
+ 言語障壁（英語報道なし）: +8点
+ サプライチェーン深度（Tier2/3）: +7点
+ 認証障壁（防衛・原子力等）: +5点
```

#### 鮮度（Freshness）

```
| decay_rate | 情報タイプ |
|------------|-----------|
| fast | 株価・需給 |
| medium | 決算・ガイダンス |
| slow | シェア・参入障壁 |
```

### C-2. 総合スコア計算

```
総合スコア = (確度スコア × 0.4) + (非対称性スコア × 4 × 0.4) + (独占力スコア × 0.2)

独占力スコア（0-100）:
- シェア90%以上: 100
- シェア70-89%: 85
- シェア50-69%: 70
- シェア30-49%: 55
- シェア20-29%: 40
- 独占的だがシェア不明: 60
```

### C-3. ランキングと選出

```
1. 総合スコアで50社をランキング
2. Top30を「投資対象リスト」として選出
3. 31-50位を「監視リスト」として保持
4. ドメインバランスの最終調整
   - いずれかのドメインがTop30で0社の場合、
     そのドメインの最上位を強制的にTop30に入れる
```

## 重要な注意事項

### やってはいけないこと

1. **30社ありきで発掘しない** — 必ず70社以上を発掘してから絞り込む
2. **AI・半導体に偏重しない** — 6ドメインから均等に発掘
3. **Tier1だけを見ない** — Tier2/3こそ非対称性が高い
4. **英語情報だけで判断しない** — 日本語情報こそ価値がある
5. **一度の検索で満足しない** — 複数の検索手法を網羅的に実行

### 成功の指標

- 一次発掘で70社以上
- スクリーニング後に50社の候補プール
- Top30の非対称性スコア16以上が10社以上
- 6ドメイン全てにTop30で最低2社以上
- Tier2/3企業をTop30の35%以上含む
- 監視リストから毎月1-2社がTop30に昇格する余地がある

### 監視リストの運用

監視リスト（31-50位）は以下の場合にTop30への昇格を検討：

1. **確度向上**: 一次資料で情報が確認された
2. **非対称性向上**: 英語メディアに報道されていない新事実発見
3. **Top30企業の降格**: シェア低下、競合台頭、情報陳腐化
4. **イベント発生**: 大型契約、M&A、政策追い風
