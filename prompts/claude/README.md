# Claude用プロンプト

## Claudeの特性

| 項目 | 評価 |
|------|------|
| 論理的推論 | ★★★★★ 優秀 |
| 矛盾検出 | ★★★★★ 優秀 |
| 一次情報源確認 | ★★★★☆ 良好 |
| 統合分析 | ★★★★★ 優秀 |
| Web検索 | ★★★☆☆ 限定的 |

## 最適なタスク

1. **矛盾解消** - Grok/Gemini出力間の矛盾を検出・解消
2. **一次情報源確認** - 市場シェア、契約等の一次情報源検証
3. **高非対称性情報確認** - 市場に認識されていない情報の検証
4. **Top 30単一ソース補強** - 重要企業の情報源多様化
5. **統合分析** - 全情報を統合してスコアリング

## プロンプト一覧

| ファイル | 用途 | 実行タイミング |
|---------|------|---------------|
| `contradiction_resolver.md` | 矛盾解消 | 自動（Phase 4） |
| `primary_source_verifier.md` | 一次情報源確認 | 自動（Phase 4） |
| `high_asymmetry_verifier.md` | 高非対称性確認 | 自動（Phase 4） |
| `top30_reinforcement.md` | Top 30補強 | 自動（Phase 8） |

## 自動実行条件

Claudeは以下の条件で自動的に補強調査を実行:

| 条件 | トリガー | アクション |
|------|---------|-----------|
| 矛盾検出 | `contradictions.json` に項目あり | 矛盾解消調査 |
| 一次情報源未確認 | 確度「要確認」の重要情報 | 一次情報源検索 |
| 高非対称性 | 非対称性スコア 15+ | 追加検証 |
| Top 30単一ソース | 重要企業の情報源が1つ | 多角的確認 |

## 実行フロー

```
Grok/Gemini出力
    ↓
Claude: 矛盾検出
    ↓
Claude: 自動補強（条件該当時）
    ↓
Claude: 統合・スコアリング
    ↓
最終レポート
```
