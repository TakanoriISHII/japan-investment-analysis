# Step 5: Top30スコアリング

## ミッション

独占マップ（静的）+ イベントログ（動的）+ 市場状態を統合し、Top30銘柄をスコアリングして4象限に配置する。

---

## 入力データ

1. `data/snapshots/monopoly_map.json` — 独占マップ（準静的）
2. `data/events.json` — イベントログ（最新イベントの影響を反映）
3. `data/snapshots/market_state.json` — 市場状態
4. `data/analysis/market_risk.json` — 市場危険度
5. `data/analysis/top30.json` — 前回スコアリング（比較用）

## 評価基準（100点満点）

### 評価軸1: グローバル独占力（30点満点）

**データソース**: monopoly_map.json → products.share

| スコア | 基準 |
|--------|------|
| 25-30 | 世界シェア50%超、または唯一の供給者 |
| 20-24 | 世界シェア30-50%、上位2社内 |
| 15-19 | 世界シェア15-30%、上位3社内 |
| 10-14 | 世界シェア10-15%、上位5社内 |
| 5-9 | 国内首位だが世界シェア10%未満 |
| 0-4 | 競争優位が不明確 |

**加点（各+2点、最大+6点）**:
- 技術的参入障壁が5年以上
- 品質認証（防衛/宇宙/原子力）が必須
- スイッチングコストが極めて高い

### 評価軸2: 資本効率改善余地（20点満点）

| スコア | 基準 |
|--------|------|
| 16-20 | ROE改善余地大 + 経営の意思表明 + 実行開始 |
| 12-15 | ROE改善余地大 + 経営の意思表明あり |
| 8-11 | ROE改善余地中 + 検討段階 |
| 4-7 | ROE業界平均程度 |
| 0-3 | ROE低迷 + 改善の意思不明 |

**イベント反映**: events.jsonで自社株買い・増配・ROE目標発表があればスコアアップ

### 評価軸3: 領域横断シナジー（25点満点）

**データソース**: monopoly_map.json → domains

| スコア | 基準 |
|--------|------|
| 21-25 | 3ドメイン以上で競争優位 + 相乗効果明確 |
| 16-20 | 2ドメインで競争優位 + 相乗効果あり |
| 11-15 | 2ドメインに関与するが片方が弱い |
| 6-10 | 1ドメインで強い競争優位 |
| 0-5 | 6ドメインとの関連が薄い |

### 評価軸4: 情報の非対称性（25点満点）

| スコア | 基準 |
|--------|------|
| 21-25 | 外国人投資家がほぼ認識していない決定的優位性 |
| 16-20 | 日本語情報の壁で過小評価 |
| 11-15 | Tier2/3で見えにくい優位性 |
| 6-10 | 一部アナリストは認識、市場全体は未反映 |
| 0-5 | グローバル投資家が十分認識 |

## 確実性・アップサイドの計算

### 確実性（certainty: 0-100）

以下を総合評価：

| 要素 | 高確実性 | 低確実性 |
|------|----------|----------|
| シェア安定性 | 過去5年で変動5%以内 | 変動大、競合台頭 |
| 契約・バックログ | 長期契約確定 | 短期・スポット |
| 地政学リスク | 中国依存20%未満 | 中国依存40%超 |
| 経営改革 | 実行中・実績あり | 未着手・停滞 |
| 規制・許認可 | 取得済み | 未承認・審査中 |

**目安**:
- 90以上: 全要素で高確実性
- 80-89: 大半で高確実性、一部リスク
- 70-79: 中程度の安定性
- 70未満: 複数のリスク要因

### アップサイド（upside: 0-100）

情報の非対称性スコア（/25）を基準に調整：

| 非対称性スコア | アップサイド目安 |
|----------------|------------------|
| 20-25 | 85-95 |
| 16-19 | 75-84 |
| 12-15 | 65-74 |
| 12未満 | 55-64 |

**調整要素**:
- 外国人保有比率30%未満 → +5
- バリュエーション割高 → -5
- 直近のポジティブイベント → +3

---

## 4象限配置

```
if (certainty >= 80 && upside >= 75) → "prime"       // 最優先投資
if (certainty >= 80 && upside < 75)  → "stable"      // 安定成長
if (certainty < 80 && upside >= 75)  → "speculative" // 投機的機会
if (certainty < 80 && upside < 75)   → "avoid"       // 回避/様子見
```

## イベントの影響反映

events.jsonの最新イベントを確認し、該当企業のスコアを調整：

| イベントタイプ | 影響 |
|----------------|------|
| 大型受注・契約 | 確実性 +5〜10 |
| シェア上昇 | 独占力スコア見直し |
| 自社株買い・増配 | 資本効率 +2〜5 |
| 中国リスク顕在化 | 確実性 -10〜20 |
| 競合参入報道 | 独占力・確実性見直し |
