**情報の確度と非対称性を投資判断の核心に据える。**

## 評価基準（100点満点）

### バフェット的視点の統合

バフェットの投資基準を評価軸に組み込む：

| バフェットの基準 | 評価軸への反映 |
|------------------|----------------|
| 自分が理解できる企業 | **Circle of Competence** — 事業理解度を評価 |
| 経営陣の質・誠実さ | **経営陣評価** — 新規軸として追加 |
| ROE 15%以上 | **資本効率** — 具体的基準を追加 |
| 負債・資本比率 33%以下 | **財務健全性** — 資本効率に統合 |
| フリーキャッシュフロー | **オーナー収益** — 資本効率に統合 |
| 競争優位の持続可能性 | **独占力** — 持続性の評価を強化 |

### 評価軸1: グローバル独占力（30点満点）★持続可能性を強化

**バフェットの「競争上の優位性（Economic Moat）」を評価：**

**データソース**: monopoly_map.json → products.share

| スコア | 基準 |
|--------|------|
| 25-30 | 世界シェア50%超、または唯一の供給者 |
| 20-24 | 世界シェア30-50%、上位2社内 |
| 15-19 | 世界シェア15-30%、上位3社内 |
| 10-14 | 世界シェア10-15%、上位5社内 |
| 5-9 | 国内首位だが世界シェア10%未満 |
| 0-4 | 競争優位が不明確 |

**加点（各+2点、最大+6点）— 持続可能性の評価**:

| 要素 | 加点 | バフェット的根拠 |
|------|------|------------------|
| 技術的参入障壁が5年以上 | +2 | 競合の追随を阻む |
| 品質認証（防衛/宇宙/原子力）が必須 | +2 | 顧客ロイヤルティの源泉 |
| スイッチングコストが極めて高い | +2 | 「ユーザーを引き付けて離さない」 |

**減点要素（新規）**:
- シェアが3年連続で低下傾向 → -3点
- 競合の技術的キャッチアップが進行中 → -2点
- 主要顧客の内製化リスク → -2点

**シェア情報の確度を確認**: share.confidence が low の場合、スコアを控えめに

### 評価軸2: 資本効率・財務健全性（20点満点）★バフェット強化

**バフェットの基準を具体化：**

| 指標 | 基準 | 根拠 |
|------|------|------|
| ROE | **15%以上** | バフェットが重視する最低ライン |
| 負債・資本比率 | **33%以下** | 適正水準、金利上昇耐性 |
| フリーキャッシュフロー | **継続的にプラス** | オーナー収益の創出 |
| 自社株買い・増配 | **実績あり** | 株主還元の意思 |

| スコア | 基準 |
|--------|------|
| 16-20 | ROE15%以上 + 負債比率33%以下 + FCF潤沢 + 還元実績 |
| 12-15 | ROE10-15% + 負債比率50%以下 + FCFプラス + 還元方針あり |
| 8-11 | ROE10%未満だが改善中 + 経営の意思表明あり |
| 4-7 | ROE業界平均程度、改善余地大 |
| 0-3 | ROE低迷 + 高負債 + 改善の意思不明 |

**イベント反映**: 
- 自社株買い発表（certainty ≥ 70）→ +2-4点
- 増配発表（certainty ≥ 70）→ +1-2点
- ROE目標公表（certainty ≥ 80）→ +2-3点
- 負債削減発表（certainty ≥ 70）→ +1-2点

### 評価軸3: 領域横断シナジー（25点満点）

**データソース**: monopoly_map.json → domains

| スコア | 基準 |
|--------|------|
| 21-25 | 3ドメイン以上で競争優位 + 相乗効果明確 |
| 16-20 | 2ドメインで競争優位 + 相乗効果あり |
| 11-15 | 2ドメインに関与するが片方が弱い |
| 6-10 | 1ドメインで強い競争優位 |
| 0-5 | 6ドメインとの関連が薄い |

**6ドメイン**: エネルギー、軍事・防衛、AI、ロボティクス、宇宙、サイバー

### 評価軸4: 情報の非対称性（25点満点）

**情報格差の3つの源泉を評価:**

| # | 源泉 | 評価ポイント | 加点 |
|---|------|--------------|------|
| 1 | 言語の壁 | 決算Q&A日本語のみ、英語報道なし | +8-10 |
| 2 | サプライチェーン深層 | Tier2/3、最終製品から不可視 | +6-8 |
| 3 | 認証・品質の参入障壁 | 防衛/宇宙/原子力の長期認定 | +5-7 |

| スコア | 基準 |
|--------|------|
| 21-25 | 3源泉全てで強い非対称性（ほぼ未認知） |
| 16-20 | 2源泉で強い非対称性（日本語情報の壁） |
| 11-15 | 1源泉で非対称性（Tier2/3等） |
| 6-10 | 一部アナリストは認識、市場全体は未反映 |
| 0-5 | グローバル投資家が十分認識済み |

**events.jsonから検証**:
- 関連イベントで `asymmetry.language_barrier: true` が多い → 加点
- 英語メディア報道があるイベント → 減点（非対称性解消）

## 確実性（Certainty）の計算

各企業について、以下を総合評価：

| 要素 | 高確実性 | 低確実性 |
|------|----------|----------|
| シェア安定性 | 過去5年で変動5%以内 | 変動大、競合台頭 |
| シェア情報の確度 | share.confidence: high | share.confidence: low |
| 契約・バックログ | 長期契約確定（certainty ≥ 85） | 短期・スポット |
| 地政学リスク | 中国依存20%未満 | 中国依存40%超 |
| 経営改革 | 実行中・実績あり | 未着手・停滞 |
| イベントの確度 | 関連イベントの平均certainty高い | 確度の低いイベントが多い |

**計算式**:
```
certainty = 基礎点(60) 
  + シェア安定性(0-10) 
  + 契約確定(0-10) 
  + 地政学リスク低(-15〜+10)
  + 経営改革(0-10)
  + イベント確度調整(-10〜+10)
```

**目安**:
- 90以上: 全要素で高確実性
- 80-89: 大半で高確実性、一部リスク
- 70-79: 中程度の安定性
- 70未満: 複数のリスク要因

### アップサイド（Upside）の計算

情報の非対称性スコア（/25）を基準に、触媒の有無で調整：

| 非対称性スコア | アップサイド基準 |
|----------------|------------------|
| 21-25 | 85-95 |
| 16-20 | 75-84 |
| 11-15 | 65-74 |
| 11未満 | 55-64 |

**調整要素**:
- 非対称性の解消トリガーが近い → +5（触媒効果）
- 外国人保有比率30%未満 → +3
- バリュエーション割高 → -5

## 4象限配置

```
if (certainty >= 80 && upside >= 75) → "prime"       // 最優先投資
if (certainty >= 80 && upside < 75)  → "stable"      // 安定成長
if (certainty < 80 && upside >= 75)  → "speculative" // 投機的機会
if (certainty < 80 && upside < 75)   → "avoid"       // 回避/様子見
```

## 最高確信銘柄の選定基準

以下の条件を全て満たす銘柄を最高確信とする：

1. **確度が最高水準**
   - シェア情報がprimary sourceで確認済み
   - 関連イベントの平均certainty ≥ 70
   - information_quality.conflictsに関連する矛盾なし

2. **非対称性が活きている**
   - asymmetry.score ≥ 15
   - 解消リスクが「低」または「中」
   - 触媒となるイベントが見込める

3. **総合スコアが上位**
   - Top3以内
   - 前回からスコア低下なし
