# Step 5: Top30スコアリング

## ミッション

**候補プール50社**を4軸評価（100点満点）でランキングし、**Top30**を選出する。
31-50位は**監視リスト**として保持し、昇格候補として追跡する。

**情報の確度と非対称性を投資判断の核心に据える。**

---

## ファネルの位置づけ

```
Step 0: 大量発掘（70-100社）→ 候補プール（50社）
    ↓
Step 5（本ステップ）:
    候補プール50社
        ↓ 4軸評価（100点満点）
    Top30選出 + 監視リスト(31-50位)
```

**重要**: Step 0では簡易スクリーニングのみ実施。Step 5で初めて詳細な財務データ（ROE、負債比率等）を収集し、正式な4軸評価を行う。

## 評価基準（100点満点）★4軸定義

### 概要

| 軸 | 配点 | 説明 |
|----|------|------|
| **独占力** | 30点 | 現在の独占度 + **5-10年の競争優位持続性** |
| **資本効率** | 20点 | **取得可能な正確な情報**に基づく財務評価 |
| **シナジー** | 25点 | 6ドメインを横断する**「不可欠性」** |
| **非対称性** | 25点 | **情報拡散時のアップサイド**（ファンダメンタルズ外） |

---

### 評価軸1: 独占力（30点満点）— 5-10年の見通しを含む

現時点のシェアだけでなく、**将来の競争優位持続性**を評価する。

| 評価項目 | 配点 | 内容 |
|----------|------|------|
| **現在のシェア・地位** | 10点 | 世界シェア、国内シェア、独占度 |
| **5-10年の競争優位持続性** | 10点 | 将来の契約パイプライン、競合の開発状況 |
| **参入障壁の強度** | 10点 | 認証期間、技術蓄積年数、スイッチングコスト |

**現在のシェア・地位（10点）**
- 世界シェア50%以上 → 8-10点
- 世界シェア30-50% → 6-8点
- 国内首位 or ニッチ独占 → 4-6点
- 上位3位以内 → 2-4点

**5-10年の競争優位持続性（10点）**
- 将来の大型契約が確定/高確率 → +3点
- 競合が5年以上キャッチアップ困難 → +3点
- 技術ロードマップで次世代も優位 → +2点
- 顧客の長期認定取得済み → +2点

**参入障壁の強度（10点）**
- 認証取得に10年以上必要（防衛・原子力） → +4点
- 認証取得に5-10年必要 → +3点
- 技術蓄積20年以上 → +3点
- スイッチングコスト極めて高い → +3点

**減点要素**:
- シェアが3年連続で低下傾向 → -3点
- 競合の技術的キャッチアップが進行中 → -2点
- 主要顧客の内製化リスク → -2点

**データソース**: monopoly_map.json → products.share
**確度確認**: share.confidence が low の場合、スコアを控えめに

---

### 評価軸2: 資本効率（20点満点）— 事実ベースで評価

**取得可能な正確な情報**のみで評価。**推定値・アナリスト予想は使わない。**

| 評価項目 | 配点 | 基準（バフェット基準） |
|----------|------|------------------------|
| **ROE** | 8点 | 15%以上で満点、10-15%で6点、10%未満で4点以下 |
| **負債・資本比率** | 6点 | 33%以下で満点、33-50%で4点、50%超で2点以下 |
| **FCF（継続性）** | 6点 | 5年連続プラスで満点、3年で4点、不安定で2点以下 |

**データソース**:
- 有価証券報告書（EDINET）
- 決算短信
- 企業IR資料

**注意**: アナリスト予想や推計値は使用しない。開示されている事実のみ。

**イベント反映（確度70以上のみ）**:
- 自社株買い発表 → +2点
- 増配発表 → +1点
- ROE目標公表（確度80以上） → +2点

---

### 評価軸3: シナジー（25点満点）— ドメイン横断の「不可欠性」

**「その技術がないと成り立たない」ドメインがいくつあるか**を評価。

#### イーロン・マスク型の統合モデル（参考）

```
インターネットインフラ ←→ AI事業 ←→ エネルギー事業
        ↓                   ↓              ↓
    宇宙事業            ロボティクス     蓄電池
        ↓                   ↓              ↓
    防衛・軍事          自動車（EV）     太陽光

→ どこでも必要不可欠な技術を持つ企業は圧倒的な優位性を持つ
```

| 評価項目 | 配点 | 内容 |
|----------|------|------|
| **ドメイン横断の不可欠性** | 15点 | その技術なしには成り立たないドメイン数 |
| **収斂による価値増幅** | 10点 | ドメイン間の相乗効果の大きさ |

**ドメイン横断の不可欠性（15点）**
- 5-6ドメインで不可欠 → 13-15点
- 3-4ドメインで不可欠 → 9-12点
- 2ドメインで不可欠 → 5-8点
- 1ドメインのみ → 1-4点

**収斂による価値増幅（10点）**
- 複数ドメインの成長が相乗的に企業価値を高める → +5点
- ドメイン間でクロスセル・技術転用が可能 → +3点
- 1つのドメインの成長が他に波及する → +2点

**6ドメイン**: エネルギー、軍事・防衛、AI、ロボティクス、宇宙、サイバー

**データソース**: monopoly_map.json → domains

---

### 評価軸4: 非対称性（25点満点）— 情報拡散時のアップサイド

独占力・資本効率で測れない**「気づかれていない価値」**を評価。

ファンダメンタルズ成長ではなく、**情報格差が解消されたときのマルチプル拡大**を狙う。

| 評価項目 | 配点 | 内容 |
|----------|------|------|
| **情報格差の存在** | 10点 | 3源泉（言語、SC深層、認証）での非対称性 |
| **情報拡散時のアップサイド** | 15点 | 「気づかれたら」どれだけ評価が変わるか |

**情報格差の存在（10点）**
- 3源泉全てで非対称性あり → 8-10点
- 2源泉で非対称性あり → 5-7点
- 1源泉で非対称性あり → 2-4点
- 広く認知済み → 0-1点

**情報拡散時のアップサイド（15点）**
- プロ（機関投資家、アナリスト）も見逃している → +5点
- 認知拡大でマルチプル2倍以上の可能性 → +5点
- 英語情報の欠如で外国人投資家が未参入 → +3点
- 解消トリガーが明確（決算、報道等） → +2点

**プロの見逃しを狙うポイント**
1. **食品/日用品企業の半導体材料** — セクター分類で見落とされる
2. **地方の中小型株** — アナリストカバレッジなし
3. **Tier3サプライヤー** — 最終製品からの可視性ゼロ
4. **認証参入障壁** — 定量化しにくいため過小評価

**events.jsonから検証**:
- 関連イベントで `asymmetry.language_barrier: true` が多い → 加点
- 英語メディア報道があるイベント → 減点（非対称性解消）

## 確実性（Certainty）の計算

各企業について、以下を総合評価：

| 要素 | 高確実性 | 低確実性 |
|------|----------|----------|
| シェア安定性 | 過去5年で変動5%以内 | 変動大、競合台頭 |
| シェア情報の確度 | share.confidence: high | share.confidence: low |
| 契約・バックログ | 長期契約確定（certainty ≥ 85） | 短期・スポット |
| 地政学リスク | 中国依存20%未満 | 中国依存40%超 |
| 経営改革 | 実行中・実績あり | 未着手・停滞 |
| イベントの確度 | 関連イベントの平均certainty高い | 確度の低いイベントが多い |

**計算式**:
```
certainty = 基礎点(60) 
  + シェア安定性(0-10) 
  + 契約確定(0-10) 
  + 地政学リスク低(-15〜+10)
  + 経営改革(0-10)
  + イベント確度調整(-10〜+10)
```

**目安**:
- 90以上: 全要素で高確実性
- 80-89: 大半で高確実性、一部リスク
- 70-79: 中程度の安定性
- 70未満: 複数のリスク要因

### アップサイド（Upside）の計算

情報の非対称性スコア（/25）を基準に、触媒の有無で調整：

| 非対称性スコア | アップサイド基準 |
|----------------|------------------|
| 21-25 | 85-95 |
| 16-20 | 75-84 |
| 11-15 | 65-74 |
| 11未満 | 55-64 |

**調整要素**:
- 非対称性の解消トリガーが近い → +5（触媒効果）
- 外国人保有比率30%未満 → +3
- バリュエーション割高 → -5

## 4象限配置

```
if (certainty >= 80 && upside >= 75) → "prime"       // 最優先投資
if (certainty >= 80 && upside < 75)  → "stable"      // 安定成長
if (certainty < 80 && upside >= 75)  → "speculative" // 投機的機会
if (certainty < 80 && upside < 75)   → "avoid"       // 回避/様子見
```

## 最高確信銘柄の選定基準

以下の条件を全て満たす銘柄を最高確信とする：

1. **確度が最高水準**
   - シェア情報がprimary sourceで確認済み
   - 関連イベントの平均certainty ≥ 70
   - information_quality.conflictsに関連する矛盾なし

2. **非対称性が活きている**
   - asymmetry.score ≥ 15
   - 解消リスクが「低」または「中」
   - 触媒となるイベントが見込める

3. **総合スコアが上位**
   - Top3以内
   - 前回からスコア低下なし
