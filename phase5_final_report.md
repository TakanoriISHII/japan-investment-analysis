## 実行手順

### Step 1: 全ファイル読み込み

4つの入力ファイルを全て読み込み、以下を確認：
- 市場危険度スコア（XX/100）
- 投資候補リスト
- Top30ランキング
- 4象限配置

### Step 2: Top10詳細分析の深掘り

Top10各銘柄について追加検索：
```
[企業名] バリュエーション PER PBR
[企業名] 外国人保有比率
[企業名] アナリスト 目標株価
```

### Step 3: シナリオ分岐策定

追加検索：
```
ドル円 予想 2025
日本株 シナリオ分析
```

3シナリオを策定：Base/Bull/Bear

### Step 4: 歴史的アナロジー

追加検索：
```
半導体装置 過去の需要サイクル
防衛費拡大 過去 日本企業 恩恵
```

### Step 5: 最高確信銘柄の特定

Top10から以下の基準で1社特定：
- 【最優先投資】象限に位置
- 情報の非対称性スコアが最高
- 反証条件の発生確率が最低
- 複数ドメインでのシナジーが明確
