## 🎯 このシステムの目的

### 情報の非対称性を武器にする

外国人投資家との情報格差を**逆手に取り**、グローバル市場が織り込んでいない日本企業の構造的優位を特定する。

```
投資 = 情報戦
・外国人投資家が見えていない情報を見つける
・確度の低い情報に基づく判断を避ける
・歴史的パターンを学び、同じ過ちを繰り返さない
```

### 情報格差の3つの源泉

| # | 源泉 | 説明 | 発見方法 |
|---|------|------|----------|
| 1 | **言語の壁** | 決算Q&A、業界紙、行政文書が日本語のみ | 英語メディアに出ていない情報を探す |
| 2 | **サプライチェーン深層** | Tier2/3の独占は最終製品から見えない | 部品・素材レベルまで掘り下げる |
| 3 | **認証・参入障壁** | 防衛・宇宙・原子力の長期認定プロセス | 認証取得企業リスト、調達実績を調査 |

## ⚠️ 市場危険度分析

**「何を買うか」の前に「今買うべきか」を判断する**

これはバフェットの教え「価値ある株が見つからないときはキャッシュポジションのまま様子をみる」に基づく。

### Tier別重みづけ（歴史的破壊力に基づく）

| Tier | 項目 | 配点 | 根拠 |
|------|------|------|------|
| **Tier1** | 金融政策転換、レバレッジ崩壊 | 各15点 | 最も破壊的（2024年8月、1990年バブル） |
| **Tier2** | バリュエーション、VIX、外国人動向 | 計30点 | 警告シグナル |
| **Tier3** | 中国リスク、地政学、流動性、為替 | 計40点 | 補助的要因 |

### バフェット的投資指針

| 市場危険度 | 解釈 | 推奨 |
|------------|------|------|
| 0-25 | 積極投資の好機 | フルポジション |
| 26-45 | 優良株を選別 | 通常ポジション |
| 46-60 | **現在（52点）** | **やや慎重（60-80%投資）** |
| 61-75 | キャッシュを増やす | ポジション縮小 |
| 76-100 | 様子見、現金化 | 大幅縮小 |

## 📈 情報の4次元評価

全ての情報を以下の4軸で評価：

| 軸 | 説明 | 例 |
|----|------|-----|
| **確度** | 噂→報道→公式発表→契約→実績 | 「検討か」(20) → 「発表」(75) → 「完了」(95) |
| **検証度** | 単独→複数ソース→一次資料確認 | SNS(不可) → 日経(中) → 政府発表(最高) |
| **非対称性** | 誰が知っているか | 日本専門家のみ(15) → ほぼ未認知(25) |
| **鮮度** | いつまで有効か | 株価(当日) → シェア(6ヶ月) → 認証(数年) |

## 🎓 設計思想（Claude Codeへの引継ぎ）

### 絶対に変更しない原則

1. **「今買うべきか」→「何を買うか」の順序**
   - 市場危険度分析が先、銘柄選定が後

2. **重みづけの根拠**
   - 歴史的破壊力に基づく（均等配分は怠慢）

3. **バフェット基準の具体値**
   - ROE 15%以上、負債比率33%以下

4. **情報品質フレームワーク**
   - 確度・検証度・非対称性・鮮度の4次元評価を維持

5. **非対称性の源泉**
   - 言語、サプライチェーン深層、認証の3源泉を常に意識

### 将来の拡張候補

- トランプ関税など**政策リスクの体系的追跡**
- **セクターローテーション**の検知
- **個別銘柄のイベントカレンダー**
- **為替シナリオ別の影響分析**の自動化
