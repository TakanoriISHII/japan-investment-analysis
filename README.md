# 日本マルチドメイン・コンバージェンス分析

## 概要

このディレクトリには、マルチドメイン・コンバージェンス投資分析フレームワークの日本版が含まれています。6つの収束ドメインにまたがるボトルネック技術を支配する日本上場企業を特定します。

## 6つのドメイン

1. **AI** — コンピュートインフラ、半導体、データセンター
2. **エネルギー** — 原子力（SMR、HALEU）、グリッドインフラ、変圧器、蓄電
3. **防衛** — ミサイル、航空機、艦船、弾薬、防衛エレクトロニクス
4. **ロボティクス** — 産業オートメーション、ドローン、UUV、ヒューマノイド
5. **宇宙** — 打上げ、衛星、月面、ISR
6. **サイバー** — 暗号化、電子戦、ISR融合、量子

## 核心テーゼ

**マルチドメイン・コンバージェンス・ギャップ**: 市場は企業を単一セクター内で評価し、クロスドメインのシナジーを見逃している。複数ドメインにサービスを提供するボトルネック技術を支配する企業は、複合的な追い風の恩恵を受ける — しかしこの価値はしばしば認識されていない。

## 評価フレームワーク（100点満点）

| 軸 | 配点 | 説明 |
|----|------|------|
| 物理的独占力 | 30 | 市場シェア、唯一の供給者としての地位、参入障壁 |
| 政策耐性 | 20 | 政策変更で恩恵を受けるか被害を受けるか |
| クロスドメイン・シナジー | 25 | 複数ドメインによって駆動される需要 |
| 情報非対称性 | 15 | 市場認識ギャップ |
| 資本効率 | 10 | FCF、希薄化リスク、バランスシート |

## クイックスタート

### 1. インテリジェンス収集（2プロンプト構造）

`prompts/external/` のプロンプトをディープリサーチLLM（GPT-4、Gemini、Perplexity）で使用:

| プロンプト | 目的 | 保存先 |
|-----------|------|--------|
| `prompt_company_info.md` | 企業発掘 + 独占情報 + 財務情報 | `raw/company_info/` |
| `prompt_broad_info.md` | 広域情報（7カテゴリ） | `raw/broad_info/` |

出力を `data/intelligence/raw/[type]/[llm名].md` に保存
- 例: `company_info/gemini.md`, `broad_info/gpt4.md`

### 2. 分析実行

Claudeに伝える: `run full analysis`

これにより:
1. 全ての生インテリジェンスを読込
2. 統合と重複排除
3. ギャップ分析を実行
4. 5軸で企業をスコアリング
5. Top 30を選定
6. レポートを生成

### 3. 出力確認

`latest/` を確認:
- `final_report.md` — 完全分析
- `executive_summary.md` — 主要な要点
- `quadrant_chart.html` — 視覚的ポジショニング

## 主要概念

### ボトルネックの問い
```
間違い: 「この企業はどのドメインに関与しているか？」
正しい: 「この技術がなくなったら何が止まるか？」
```

### 攻守バランス
市場リスクスコアがポジショニングの積極性を決定:
- 低リスク (0-25): 非対称性を最大化、不確実性を許容
- 中リスク (26-50): 確実性とアップサイドのバランス
- 高リスク (51+): 確実性を優先、投機的ポジションを縮小

### 4象限マトリックス

| 象限 | 確実性 | アップサイド | アクション |
|------|--------|-------------|----------|
| 最優先 | 高 | 高 | コアポジション |
| 安定成長 | 高 | 低 | 分散保有 |
| 投機的 | 低 | 高 | 小ポジション、カタリスト待ち |
| 回避 | 低 | 低 | ポジションなし |

## 情報収集の7カテゴリ

| # | カテゴリ | 内容 |
|---|---------|------|
| A | 地政学 | 日中関係、台湾、輸出規制、経済安全保障 |
| B | 市場リスクシグナル | 日銀政策、円相場、バリュエーション |
| C | メガトレンド | AI設備投資、エネルギー転換、防衛支出 |
| D | 6ドメイン成長 | 市場規模、成長率、投資動向 |
| E | 大型投資パイプライン | 経産省補助金、GX投資、防衛費増額 |
| F | 見逃しスキャン | 6ドメイン外の新興領域 |
| G | その他 | 金利、為替、センチメント |
