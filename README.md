# マクロパルス × マルチドメイン・コンバージェンス分析

日本企業のグローバル競争優位を分析し、投資判断に資するレポートを自動生成するシステムです。

## 特徴

- **5フェーズ分割方式** — コンテキスト圧縮を回避し、高精度な分析を実現
- **Claude Code + GitHub連携** — 一気通貫実行とバージョン管理
- **情報の非対称性に着目** — 外国人投資家が見落としている日本企業の優位性を発見

## クイックスタート

### 1. リポジトリをクローン
```bash
git clone https://github.com/[your-username]/japan-investment-analysis.git
cd japan-investment-analysis
```

### 2. Claude Codeで開く
```bash
claude
```

### 3. フル分析を実行
Claude Codeのプロンプトで：
```
フル分析を実行
```

## 分析フロー

```
Phase 1: 市場危険度スコア（10-15分）
    ↓
Phase 2: グローバル独占マップ（20-30分）
    ↓
Phase 3: マクロパルス情報収集（20-30分）
    ↓
Phase 4: Top30スコアリング（15-20分）
    ↓
Phase 5: 最終レポート統合（20-30分）
```
