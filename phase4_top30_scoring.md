## 評価基準（100点満点）

### 評価軸1：グローバル独占力（30点満点）

| スコア | 基準 |
|--------|------|
| 25-30点 | 世界シェア50%超、または特定工程で唯一の供給者 |
| 20-24点 | 世界シェア30-50%、上位2社内 |
| 15-19点 | 世界シェア15-30%、上位3社内 |
| 10-14点 | 世界シェア10-15%、上位5社内 |
| 5-9点 | 国内首位だが世界シェア10%未満 |
| 0-4点 | 競争優位が不明確 |

加点（各+2点、最大+6点）：技術的参入障壁5年以上、品質認証必須、スイッチングコスト高

### 評価軸2：資本効率改善余地（20点満点）

| スコア | 基準 |
|--------|------|
| 16-20点 | ROE改善余地大＋経営の意思表明＋実行開始 |
| 12-15点 | ROE改善余地大＋経営の意思表明あり |
| 8-11点 | ROE改善余地中＋検討段階 |
| 4-7点 | ROE業界平均程度 |
| 0-3点 | ROE低迷＋改善の意思不明 |

### 評価軸3：領域横断シナジー（25点満点）

| スコア | 基準 |
|--------|------|
| 21-25点 | 3ドメイン以上で競争優位＋相乗効果明確 |
| 16-20点 | 2ドメインで競争優位＋相乗効果あり |
| 11-15点 | 2ドメインに関与するが片方が弱い |
| 6-10点 | 1ドメインで強い競争優位 |
| 0-5点 | 6ドメインとの関連が薄い |

### 評価軸4：情報の非対称性（25点満点）

| スコア | 基準 |
|--------|------|
| 21-25点 | 外国人投資家がほぼ認識していない決定的優位性 |
| 16-20点 | 日本語情報の壁で過小評価 |
| 11-15点 | Tier2/3で見えにくい優位性 |
| 6-10点 | 一部アナリストは認識、市場全体は未反映 |
| 0-5点 | グローバル投資家が十分認識 |

## 4象限配置基準

### 横軸：確実性
**高確実性**: 長期契約確定、規制取得済み、シェア安定、改善実績あり
**低確実性**: 技術未実証、規制未承認、競争激化リスク、改革未着手

### 縦軸：アップサイド
**高アップサイド**: 非対称性16点以上、認識ギャップ大
**低アップサイド**: 非対称性10点以下、既に認識済み
