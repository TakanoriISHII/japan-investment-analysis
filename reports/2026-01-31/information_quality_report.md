## 4. Grokデータ品質問題の詳細

### 問題概要

| 項目 | 詳細 |
|------|------|
| **アラートID** | CHG-012 |
| **問題種別** | data_quality_alert |
| **検出フェーズ** | Phase 4-1（矛盾解消） |
| **確度** | A（問題の存在自体は確定） |

### 問題の詳細

Phase 4-1の矛盾解消プロセスにおいて、Grokが収集した企業データに**系統的コピーエラー**が検出された。具体的には以下の問題が確認された:

1. **安川電機（6506.T）のデータ**: 一部の財務指標にハーモニック・ドライブ・システムズ（6324.T）のデータが混入
2. **ハーモニック・ドライブ・システムズ（6324.T）のデータ**: 日立製作所（6501.T）の売上規模データが一部混在
3. **日立製作所（6501.T）のデータ**: 株式分割前後のデータが混在し、時価総額に不整合

### 根本原因分析

Grokのデータ収集において、類似ドメイン（ロボティクス/産業機器）の企業間でコンテキスト情報が混在する傾向が確認された。これは単発のエラーではなく、**系統的（systematic）なパターン**であり、以下の特徴を持つ:

- 同一ドメイン内の企業間でデータが「スライド」する
- 特に数値データ（売上、時価総額、シェア率）に混在が発生しやすい
- テキスト情報（事業説明、戦略）には混在が少ない

### 対応措置

| 措置 | 内容 | ステータス |
|------|------|-----------|
| 重み引き下げ | Grok単一情報源データの重みを50% → 35%に引き下げ | 実施済み |
| 影響範囲特定 | 約15社のGrok-only財務データを特定 | 実施済み |
| クロスバリデーション | 影響企業の全数値データを他LLM/一次情報源で再検証 | 実施済み |
| 確度調整 | 全体確度スコアに-15%信頼度調整を適用 | 反映済み |

### 次回アクション

- [ ] Grok収集プロンプトに**出力検証ステップ**を追加（企業名・ティッカーの再確認ステップ）
