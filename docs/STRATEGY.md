## 基本理念

**情報は不確実性を減少させる。**

投資で勝つために:

1. **情報精度の最大化** — 噂と事実を区別し、一次情報源で検証する
2. **情報の非対称性を活用** — 他者が持たない知識で優位性を得る
3. **情報の鮮度を管理** — 古いデータに基づく意思決定を排除する

### 投資哲学の核心

```
正しい順序:
1. 今買うべきか？（市場リスク）
2. 何を買うか？（企業評価）
3. いくら買うか？（ポジションサイズ）

間違った順序:
1. この企業が好き → 2. 市場環境を無視 → 3. 全力投資
```

## ミッション定義

### 目的

**マルチドメイン・コンバージェンスにおける情報の非対称性を活用する** — 市場が単一ドメインに焦点を当てることで過小評価している、複数ドメインの収束点で構造的優位性を持つ日本企業を特定する。

### 核心的洞察: マルチドメイン・コンバージェンス・ギャップ

| # | 情報非対称性の源泉 | 説明 | 活用方法 |
|---|-------------------|------|---------|
| 1 | **単一ドメイン集中** | 市場アナリストは一つのセクターに特化し、クロスドメインのシナジーを見逃す | 3つ以上のドメインに不可欠な技術を持つ企業を探す |
| 2 | **サプライチェーンの深さ** | Tier 2/3のボトルネック供給者は最終製品からは見えない | 部品・材料レベルまで掘り下げる |
| 3 | **認証障壁** | 防衛/原子力/航空宇宙の認証が見えないモートを形成 | 認証保有者、調達記録を調査 |
| 4 | **政策の複雑性** | 日中関係、経済安全保障推進法、経産省支援策は複雑で変化が速い | 一次情報源を追跡: 官報、経産省公表資料 |

### 視点

**「グローバル・ボトルネック・コントロール」** — コンピュート、電力、素材、精密部品、製造装置における重要ノードを支配する日本企業は、産業が収束する中で代替不可能なレバレッジを持つ。

### 評価の核心的問い

```
間違い: 「この企業はどのドメインに関与しているか？」

正しい問い:
- 「この技術がなくなったら何が止まるか？」
- 「これなしでは機能できない企業/産業はいくつあるか？」
- 「競合が追いつくまで何年かかるか？」
```

## 6つのドメイン

### ドメイン定義

| # | ドメイン | 主要領域 | 日本企業の強み |
|---|---------|---------|---------------|
| 1 | **エネルギー** | 原子力、グリッド、変圧器、蓄電、水素 | 原子力技術、蓄電池、水素サプライチェーン |
| 2 | **防衛** | ミサイル、航空機、艦船、弾薬、防衛エレクトロニクス | 精密誘導、電子機器、艦船建造 |
| 3 | **AI** | コンピュートインフラ、半導体製造装置、データセンター | 半導体製造装置、素材、パワー半導体 |
| 4 | **ロボティクス** | 産業用、ドローン、UUV、ヒューマノイド | FA機器、産業用ロボット、精密部品 |
| 5 | **宇宙** | 打上げ、衛星、月面、ISR | H3ロケット、衛星部品、宇宙機器 |
| 6 | **サイバー** | 暗号化、電子戦、ISR融合 | セキュリティソフト、通信機器 |

### 収束構造

```
AI → 電力需要 → エネルギー
防衛 → 宇宙ISR → サイバー
全ドメイン → 半導体が必要 → AI/エネルギー
```

### 収束マトリックス

|          | エネルギー | 防衛 | AI | ロボティクス | 宇宙 | サイバー |
|----------|:--------:|:----:|:--:|:----------:|:----:|:------:|
| エネルギー | - | 基地電力 | DC電力 | 製造電力 | 推進 | グリッド防御 |
| 防衛 | 基地電力 | - | 自律兵器 | ドローン | ISR | 電子戦 |
| AI | DC電力 | 自律兵器 | - | 制御 | 衛星AI | 脅威検知 |
| ロボティクス | 製造電力 | ドローン | 制御 | - | 宇宙ロボ | 物理侵入 |
| 宇宙 | 推進 | ISR | 衛星AI | 宇宙ロボ | - | 衛星通信 |
| サイバー | グリッド防御 | 電子戦 | 脅威検知 | 物理侵入 | 衛星通信 | - |

### 核心テーゼ

**経済安全保障と技術自立が6ドメイン全てへの投資を同時に加速させている。複数ドメインにまたがるボトルネック技術を支配する日本企業は、複合的な需要増加を経験する。**

## ドメイン管理

### 新ドメイン昇格基準

「見落としスキャン」で発見された新興領域は、以下の**全て**を満たす場合、正式ドメインに昇格する:

| 基準 | 閾値 | 検証方法 |
|------|------|---------|
| 市場規模 | TAM $500B以上 または 年成長率30%以上 | 業界レポート、政府予測 |
| 収束性 | 既存ドメインとの収束ポイントが2つ以上 | 収束マトリックス分析 |
| 投資可能性 | 3四半期連続で候補企業が5社以上発見 | 企業スキャン結果 |
| ボトルネック存在 | 独立したボトルネック企業が存在 | 独占力分析 |

### ドメイン昇格プロセス

```
1. 見落としスキャンで候補発見
   ↓
2. 昇格基準の評価（四半期ごと）
   ↓
3. 基準充足 → 候補ドメインとして3四半期追跡
   ↓
4. 3四半期連続で基準維持 → 正式ドメインに昇格
   ↓
5. 収束マトリックス更新、評価フレームワーク調整
```

### ドメイン統合・降格基準

以下の**いずれか**を満たす場合、統合または降格を検討:

| 基準 | 閾値 | アクション |
|------|------|-----------|
| ボトルネック消失 | 独立したボトルネック企業が存在しない | 他ドメインに統合 |
| 活動停滞 | 2四半期連続で重要イベントなし | 降格検討 |
| 収束性低下 | 収束ポイントが1つ以下に減少 | 降格検討 |
| 投資不能 | 候補企業が2社未満に減少 | 降格または統合 |

## クロスマーケット統合戦略【v3.0.0〜】

### 原則

**日米2リポジトリの独立性を維持しつつ、分析完了後にクロスマーケットの情報を双方向にプッシュする。**

### 目的

1. **相対比較**: 同一サプライチェーン内のJP vs US企業を比較（例: TEL vs Lam Research）
2. **先行指標**: 一方の市場のイベントを他方の先行指標として活用
3. **グローバル・ドメイン収束**: 6ドメインのグローバル全体像を把握
4. **情報の非対称性最大化**: 単一市場では見えない構造的優位性を発見

### データフロー

```
JP分析完了（Phase 9.5）
  → cross_market_summary.json生成
  → USリポジトリのdata/cross_market/incoming/にコピー

US分析完了（Phase 9.5）
  → cross_market_summary.json生成
  → JPリポジトリのdata/cross_market/incoming/にコピー
```

### サプライチェーンリンク

日米間のサプライチェーン関係を`supply_chain_links.json`で管理:
- 競合関係（TEL vs LRCX, MHI vs RTX等）
- 顧客-サプライヤー関係
- 技術依存関係（日本素材 → US半導体等）

### クロスマーケット活用の判断基準

| 状況 | 活用方法 |
|------|---------|
| JP企業のUS競合が決算サプライズ | JP企業の需要環境の先行指標として活用 |
| US政策がJPサプライチェーンに影響 | JP企業の軸5（政策・触媒）を調整 |
| 日米のPER格差が拡大 | 相対的な投資魅力度を比較 |
| 同一ドメインでJP > US（またはその逆） | グローバル配分の参考情報 |

## エグジット戦略【v3.0.0〜】

### 基本方針

**「買い」の判断だけでなく「売り」の判断も体系化する。**

> Buffett: "第一のルール: 損をするな。第二のルール: 第一のルールを忘れるな。"
> Soros: "ナラティブが崩壊したら、即座に撤退せよ。"

### ランクベースアラート

| ランク変動 | アラート | 推奨アクション |
|:--------:|:------:|-------------|
| < 5 低下 | なし | 通常保有 |
| 5 低下 | イエロー | 低下要因分析、ポジション見直し |
| ≥ 10 低下 | レッド | テーゼ無効化精査、ポジション縮小推奨 |

### 市場リスク連動のキャッシュ管理

| 市場リスク | キャッシュ目標 | ポジション規模 |
|:---------:|:-----------:|:------------:|
| 0-25（積極） | 5-10% | 全体100% |
| 26-50（バランス） | 10-15% | 全体100% |
| 51-75（保守） | 15-25% | 全体80% |
| 76-100（防御） | 25-40% | 全体50% |
