## 評価の順序

**「何を買うか」の前に「今買うべきか」を決める。**

```
┌─────────────────────────────────────────────────────────────┐
│ Step 1: 市場リスク分析                                      │
│         → 今は投資すべきタイミングか？                      │
│         → 攻め/守りのスタンスを決定                         │
├─────────────────────────────────────────────────────────────┤
│ Step 2: 企業評価                                            │
│         → どの企業が最も優位か？                            │
│         → 5軸スコアリング                                   │
├─────────────────────────────────────────────────────────────┤
│ Step 3: 4象限配置                                           │
│         → 確実性 × アップサイドでポジション決定             │
├─────────────────────────────────────────────────────────────┤
│ Step 4: シナリオ分析                                        │
│         → ベース/ブル/ベアでストレステスト                  │
├─────────────────────────────────────────────────────────────┤
│ Step 5: パフォーマンス検証（事後）                          │
│         → 予測は当たったか？フレームワーク改善              │
└─────────────────────────────────────────────────────────────┘
```

## 市場リスク分析

### ティア加重リスクカテゴリー（合計100点）

**根拠**: 全てのリスクが同等ではない。歴史的証拠は、日銀の政策転換と円相場変動が日本市場に最も大きな影響を与える。

#### ティア1: 最高インパクト（35点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 1 | **金融政策リスク** | 20 | 日銀政策転換（YCC修正、利上げ）、財政出動 | 2022年YCC修正、2024年マイナス金利解除 |
| 2 | **バリュエーション過熱** | 15 | 日経平均PER、TOPIX PBR、NT倍率 | 1989年バブル、2024年史上最高値 |

#### ティア2: 高インパクト（34点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 3 | **為替リスク** | 12 | ドル円水準、円相場ボラティリティ、日米金利差 | 2022年円安（150円突破）、介入リスク |
| 4 | **海外投資家動向** | 12 | 海外投資家売買動向、先物ポジション | 海外勢の売り越し/買い越しトレンド |
| 5 | **センチメント過熱** | 10 | 日経VI、騰落レシオ、信用評価損率 | 逆張り指標 |

#### ティア3: 中程度インパクト（31点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 6 | **外部ショック** | 11 | 地政学（中国・台湾）、サプライチェーン | 米中摩擦、半導体規制、台湾有事リスク |
| 7 | **流動性リスク** | 10 | JGB市場、日銀保有比率、社債市場 | 日銀YCC限界、JGB流動性枯渇 |
| 8 | **業績乖離** | 10 | 株価とEPS成長のギャップ、為替前提乖離 | 想定為替レートとの乖離 |

### 歴史的クラッシュパターン（毎回確認）

| パターン | 主因 | 先行指標 | ティア |
|---------|------|---------|-------|
| 日銀政策転換 | 金融緩和終了/利上げ | 日銀会合シグナル、植田総裁発言 | **ティア1** |
| バリュエーション・クラッシュ | PER/PBRの極端な超過 | 日経平均PER > 18、PBR > 1.5 | **ティア1** |
| 円高ショック | 急激な円高進行 | 日米金利差縮小、介入観測 | ティア2 |
| 海外勢売りアンワインド | 海外投資家の大量売却 | 先物売り越し、現物売り越し継続 | ティア2 |
| 流動性危機 | JGB市場機能不全 | 日銀指値オペ、社債スプレッド拡大 | ティア3 |
| 外部ショック | 地政学リスク、災害 | 台湾有事、震災、パンデミック | ティア3 |

### リスクスコア解釈と推奨スタンス

| スコア | 解釈 | 推奨スタンス | ターゲット企業 |
|--------|------|-------------|---------------|
| 0-25 | 低リスク | アグレッシブ | 高非対称性 + 高シナジーの日本企業 |
| 26-50 | 中リスク | バランス | 確実性とアップサイドの両立 |
| 51-75 | 高リスク | 保守的 | 高独占 + 高政策耐性 |
| 76-100 | 極端リスク | ディフェンシブ | 最高確信銘柄のみ、現金比率増 |

## 企業評価: 5軸フレームワーク

### 相対評価の原則（重要）

```
┌─────────────────────────────────────────────────────────────────────┐
│                5軸評価は相対的かつ将来予測的である                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  物理的独占力、政策耐性、シナジーは極めて重要だが、                  │
│  これらは「絶対的」ではなく「相対的」に評価する。                    │
│  さらに「現在の事実」だけでなく「将来の不確実性」も考慮する。       │
│                                                                     │
│  考慮すべき要因:                                                     │
│  1. 【相対】競合・ライバルの動向（追い上げ/撤退/新規参入）          │
│  2. 【相対】情勢の変化（政策転換/技術革新/市場構造変化）            │
│  3. 【相対】前回評価からの変化（改善/悪化/停滞）                    │
│  4. 【将来】競合が追いつくまでの時間軸（2年/5年/10年+）             │
│  5. 【将来】技術的モートの持続性（代替技術の有無）                  │
│  6. 【将来】投資パイプラインの確定度（構造的/循環的）               │
│                                                                     │
│  【評価手順】                                                        │
│  Step 1: ベーススコアを算出（絶対的な位置）                          │
│  Step 2: 相対調整（競合動向、情勢変化、前回比）                      │
│  Step 3: 将来性調整（技術的持続性、政策持続性、需要持続性）          │
│  Step 4: 最終スコアを決定                                            │
│                                                                     │
│  例: 東京エレクトロン 半導体製造装置シェア高                         │
│  - ベース: 28点（シェア50%以上の領域あり）                           │
│  - 相対: ASML等と競争 → -2点                                         │
│  - 将来: 技術優位は継続 → +1点                                       │
│  - 最終: 27点                                                        │
│                                                                     │
│  例: 三菱重工 防衛用エンジン国内独占                                 │
│  - ベース: 30点（国内唯一の供給者）                                  │
│  - 相対: 競合なし → ±0点                                            │
│  - 将来: 防衛費増額で需要増、代替なし → +3点                         │
│  - 最終: 33点（上限超過は30点に調整）                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 5軸概要（100点満点）

| 軸 | 配点 | 核心的問い |
|----|------|-----------|
| **物理的独占力** | 30 | この企業なしでXはできるか？ |
| **政策耐性** | 20 | 政策変更で恩恵を受けるか、被害を受けるか？ |
| **クロスドメイン・シナジー** | 25 | 何ドメインの成長がこの需要を押し上げるか？ |
| **情報非対称性** | 15 | このボトルネック・ポジションは市場に認識されているか？ |
| **資本効率** | 10 | 株主価値を破壊せず成長を資金調達できるか？ |

---

### 軸1: 物理的独占力（30点）

#### ベーススコア（絶対的位置）

| スコア | 基準 |
|--------|------|
| 25-30 | 50%以上の市場シェア または 重要領域での唯一の供給者 |
| 20-24 | 30-50%シェア、トップ2プレイヤー |
| 15-19 | 15-30%シェア、トップ3プレイヤー |
| 10-14 | 10-15%シェア、トップ5プレイヤー |
| 5-9 | 競争優位性は存在するが10%未満のシェア |
| 0-4 | 競争優位性が不明確 |

#### 相対調整（競合・変化・将来性を考慮）

| 条件 | 調整 | 例 |
|------|------|-----|
| 競合がシェア拡大中 | -1〜-3 | 海外勢が国内市場を侵食 |
| 競合が撤退/弱体化 | +1〜+2 | 競合の事業撤退発表 |
| 新規参入者の脅威 | -1〜-3 | 中国企業の台頭、スタートアップ参入 |
| 参入障壁が強化 | +1〜+2 | 新規制で認証ハードル上昇 |
| 前回比シェア低下 | -1〜-2 | 市場シェアの縮小トレンド |
| 前回比シェア拡大 | +1〜+2 | 市場シェアの拡大トレンド |

#### 将来性調整（技術的持続性）

| 条件 | 調整 | 例 |
|------|------|-----|
| 競合が2-3年で追いつく見込み | -2〜-4 | 中国・韓国勢が技術的に追い上げ |
| 競合が5年以上追いつけない | +2〜+3 | 半導体製造装置、素材技術の独占 |
| 技術的モートが縮小傾向 | -1〜-3 | オープンソース化、標準化の進行 |
| 技術的モートが拡大傾向 | +1〜+3 | 新特許、独自規格の普及拡大 |
| 代替技術の台頭リスク | -2〜-4 | 量子コンピューティング、新材料 |
| 代替技術なし（唯一の解決策） | +2〜+3 | 原子力認証、特定周波数帯ライセンス |

**評価質問:**
- この企業の製品が入手できなくなったら何が壊れるか？
- Fortune 500企業のうち何社が顧客か？
- 業界全体（競合を含む）がこれに依存しているか？
- **【相対】** 競合は追いついてきているか、差は広がっているか？
- **【相対】** 新規参入の動きはあるか、参入障壁は維持されているか？
- **【将来】** 競合が技術的に追いつくまで何年かかるか？
- **【将来】** 代替技術が出現する可能性はあるか？

### 軸2: 政策耐性（20点）

#### ベーススコア（現在の政策環境）

| スコア | 基準 |
|--------|------|
| 17-20 | 極めて耐性あり；政策変更がより多くの需要を生む |
| 13-16 | 高耐性；政策リスク最小 |
| 9-12 | 中立；政策変更の影響は混在 |
| 5-8 | 政策変更に対してやや脆弱 |
| 0-4 | 潜在的な政策変更から高リスク |

#### 相対調整（情勢変化を考慮）

| 条件 | 調整 | 例 |
|------|------|-----|
| 政策の風向きが好転 | +1〜+3 | GX投資追加、防衛費増額決定 |
| 政策の風向きが悪化 | -1〜-3 | 補助金削減案、規制強化の動き |
| 超党派支持が強化 | +1〜+2 | 両党からの支持表明 |
| 政権交代リスク増大 | -1〜-2 | 選挙での政策転換リスク |
| 国際情勢で重要性増 | +1〜+2 | 地政学リスクで国内調達重視 |
| 国際情勢で重要性減 | -1〜-2 | 緊張緩和で優先度低下 |

#### 将来性調整（政策の持続性）

| 条件 | 調整 | 例 |
|------|------|-----|
| 長期契約で収益確定 | +2〜+3 | 長期電力契約、10年防衛契約 |
| 短期予算依存 | -1〜-2 | 単年度予算、継続審議中 |
| 法制化済み（撤回困難） | +1〜+2 | 経済安全保障推進法指定、GX投資枠組み |
| 行政命令のみ（撤回容易） | -1〜-2 | 次期政権で撤回可能 |

**耐性指標:**

```
プラス要因:
+ 防衛大綱で義務付けられたプログラム（削減困難）
+ 与野党支持（超党派で支持）
+ 経済安全保障推進法の重要物資指定
+ 国内調達要件の受益者
+ 長期契約締結済み

マイナス要因:
- 政権交代で優先度低下リスク
- 特定省庁予算に依存
- 輸出規制の対象
- 環境許認可の脆弱性
```

**【相対】評価質問:**
- 前回と比べて政策環境は好転/悪化しているか？
- 今後6ヶ月で政策変更の可能性はあるか？

### 軸3: クロスドメイン・シナジー（25点）

| 要素 | 配点 | 基準 |
|------|------|------|
| **ドメイン数** | 15 | 何ドメインの成長が需要を増加させるか？ |
| **シナジー効果** | 10 | ドメイン同士が相乗効果を生むか？ |

#### ドメイン数（15点）

各ドメインについて「[ドメイン]への投資が増加したら、この技術への需要は増加するか？」と問う。

```
[ ] AI投資増加 → 需要増加？        → +2.5点
[ ] エネルギー投資増加 → 需要増加？ → +2.5点
[ ] 防衛投資増加 → 需要増加？      → +2.5点
[ ] ロボティクス投資増加 → 需要増加？ → +2.5点
[ ] 宇宙投資増加 → 需要増加？       → +2.5点
[ ] サイバー投資増加 → 需要増加？   → +2.5点

最大15点（全6ドメインで需要増加）
```

**重要**: 「この技術への需要が増加する」を評価、「企業がドメインに参加している」ではない。

#### シナジー効果（10点）

| スコア | 基準 |
|--------|------|
| 10 | 複数ドメインの成長が技術価値を指数的に増加 |
| 7 | ドメイン間で明確なクロスセル/技術移転 |
| 4 | 一つのドメインの成長が別のドメインの需要に波及 |
| 2 | 波及効果は限定的 |
| 0 | クロスドメイン・シナジーなし |

#### 相対調整（ドメイン成長率の変化を考慮）

| 条件 | 調整 | 例 |
|------|------|-----|
| 主要ドメインの成長加速 | +1〜+3 | AI投資がデータセンター建設等で急増 |
| 主要ドメインの成長減速 | -1〜-3 | 防衛費削減、エネルギー投資縮小 |
| 新たなドメイン連携が出現 | +1〜+2 | AI×エネルギーの融合加速 |
| ドメイン間の連携が弱まる | -1〜-2 | 特定ドメインの独立化 |

#### 将来性調整（ドメイン成長の持続性）

| 条件 | 調整 | 例 |
|------|------|-----|
| 5年以上の投資パイプライン確定 | +2〜+3 | GX投資150兆円、防衛費43兆円（5年） |
| 投資が一過性のブーム | -1〜-3 | 補助金終了後の需要減リスク |
| 構造的需要増（不可逆） | +1〜+2 | 電力需要増、データセンター必須化 |
| 循環的需要（景気依存） | -1〜-2 | 設備投資サイクルに連動 |

**【相対・将来】評価質問:**
- 前回と比べて各ドメインの成長率は加速/減速しているか？
- 新たなドメイン間連携は生まれているか？
- **【将来】** 投資パイプラインは何年分確定しているか？
- **【将来】** 需要は構造的か、循環的か？

### 軸4: 情報非対称性（15点）

| 要素 | 配点 | 基準 |
|------|------|------|
| **認識ギャップ** | 9 | この優位性はどの程度見落とされているか？ |
| **認識トリガー** | 6 | より広い認知を促すカタリストがあるか？ |

#### 認識ギャップ（9点）

| スコア | 基準 |
|--------|------|
| 8-9 | 機関投資家とアナリストが完全に見落としている |
| 6-7 | セクター専門家は認識、より広い市場は未認識 |
| 4-5 | 認識されているが過小評価 |
| 2-3 | 広く知られているが、一部過小評価 |
| 0-1 | 完全に認識され織り込み済み（NVIDIAなど） |

#### 認識トリガー（6点）

| スコア | 基準 |
|--------|------|
| 5-6 | 6ヶ月以内に明確なトリガー（契約発表、決算、カンファレンス） |
| 3-4 | トリガーは存在するがタイミング不明 |
| 1-2 | 明確なトリガーなし、段階的な認識を予想 |
| 0 | すでに認識済みまたは認識への道筋なし |

### 軸5: 資本効率（10点）

| 要素 | 配点 | 基準 |
|------|------|------|
| **FCF品質** | 5 | フリーキャッシュフローの持続性 |
| **希薄化リスク** | 3 | 株式報酬、エクイティ調達 |
| **バランスシート** | 2 | 債務の持続性 |

#### FCF品質（5点）

| スコア | 基準 |
|--------|------|
| 5 | FCFプラス、成長中、設備投資と還元を賄う |
| 4 | FCFプラス、持続可能 |
| 3 | FCFブレークイーンまたは微プラス |
| 2 | FCFマイナスだがプラス化への明確な道筋（高成長フェーズ） |
| 1 | FCFマイナス、プラス化への道筋不明確 |
| 0 | FCF大幅マイナス、明確な道筋なし |

#### 希薄化リスク（3点）

| スコア | 基準 |
|--------|------|
| 3 | SBC最小限、エクイティ調達予定なし、自社株買い |
| 2 | 中程度のSBC（年5%未満）、エクイティ調達なし |
| 1 | 高SBCまたは潜在的エクイティ調達 |
| 0 | 希薄化調達の履歴、SBC年10%超 |

#### バランスシート（2点）

| スコア | 基準 |
|--------|------|
| 2 | ネットキャッシュ または Debt/EBITDA 2x未満 |
| 1 | Debt/EBITDA 2-4x、管理可能 |
| 0 | Debt/EBITDA 4x超 または ディストレスリスク |

#### 高成長企業への調整ルール

| 条件 | 調整 |
|------|------|
| 物理的独占力 ≥ 25 かつ 売上YoY +30% | FCFマイナスへのペナルティ軽減 |
| 物理的独占力 < 20 かつ FCFマイナス | フルペナルティ適用 |

## 情報鮮度調整（補助メカニズム）

### 5軸相対評価との関係

```
┌─────────────────────────────────────────────────────────────────────┐
│                    スコアリングの2層構造                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【主】5軸相対評価（100点満点）                                      │
│  - 各軸でベーススコア + 相対調整（競合・情勢・前回比）               │
│  - 競合の追い上げ、政策変化、ドメイン成長率の変化を反映             │
│  → これが評価の本体。相対的な変化が常に反映される。                 │
│                                                                     │
│  【補助】情報鮮度調整（-10〜+15点）                                  │
│  - 新規発見、情報陳腐化など「情報管理」の観点からの調整              │
│  → stable/蓄積量による有利/不利を調整する補助メカニズム             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 情報鮮度調整項目

```
┌─────────────────────────────────────────────────────────────────────┐
│  情報鮮度調整（-10〜+15点）                                          │
│  + 新規発見ボーナス（+5〜+10）: 市場未認識の企業                     │
│  + 不確実性減少（+2〜+5）: 噂→確定に昇格                            │
│  - 情報陳腐化（-5〜-10）: stable/の長期未更新                        │
│  - 変化なし長期化（-3〜-5）: 3回以上変動なし                         │
└─────────────────────────────────────────────────────────────────────┘
```

### 動的調整項目

#### 1. 新規発見ボーナス（+5〜+10点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 今回初めて発見した企業 | +10 | 市場がまだ認識していない可能性大 |
| 過去3回以内に発見した企業 | +5 | まだ認知度が低い |
| stable/に6ヶ月以上蓄積 | 0 | すでに評価が安定 |

#### 2. 重要変化検知（+3〜+5点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 変化アラート発生（市場シェア、モート） | +5 | 市場が追いついていない変化 |
| 新規大型契約（$100M+） | +5 | カタリスト |
| 経営層変更、M&A | +3 | 潜在的転機 |

#### 3. 不確実性減少（+2〜+5点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 噂→公式発表に昇格 | +5 | リスク大幅減 |
| 未確認→確認に昇格 | +3 | 信頼度向上 |
| 一次情報源で検証完了 | +2 | 確度確定 |

#### 4. 情報陳腐化（-5〜-10点）

| 条件 | ペナルティ | 理由 |
|------|-----------|------|
| stable/の最終更新が6ヶ月以上前 | -5 | 情報の鮮度懸念 |
| stable/の最終更新が12ヶ月以上前 | -10 | 大幅な陳腐化リスク |
| 今回LLM収集で言及なし | -3 | 関心低下の可能性 |

#### 5. 変化なし長期化（-3〜-5点）

| 条件 | ペナルティ | 理由 |
|------|-----------|------|
| 3回連続でスコア変動なし | -3 | 動きがない=機会が少ない |
| 6回連続でスコア変動なし | -5 | 硬直化、見直し必要 |

### 適用例

```
例1: 新規発見の小型企業
- 5軸ベーススコア: 65点（情報不足で控えめ評価）
- 新規発見ボーナス: +10
- 重要変化検知: +5（大型契約発表）
→ 最終スコア: 80点

例2: stable/に長期蓄積の大型企業
- 5軸ベーススコア: 85点（詳細情報で高評価）
- 情報陳腐化: -5（8ヶ月更新なし）
- 変化なし長期化: -3
→ 最終スコア: 77点

→ 新規発見企業が大型企業を逆転する可能性あり
```

## 情報品質評価: 4次元フレームワーク

全ての情報を以下の4軸で評価する。

### 次元1: 確実性（この情報はどの段階か？）

| 段階 | スコア | 定義 | 例 |
|------|--------|------|-----|
| 噂 | 10-20 | 推測、匿名情報源、「検討中」 | "検討中と報道" |
| 未確認報告 | 30-40 | 報告されたが一次情報源未確認 | "発表予定" |
| 確認報告 | 50-60 | 複数報告で確認、公式情報源なし | "最終交渉中" |
| 公式発表 | 70-80 | 公式発表が存在 | "本日発表" |
| 契約締結 | 85-95 | 契約締結、予算成立、法制化 | "$X契約を獲得" |
| 実行済 | 95-99 | 完了、納品済、決算計上済 | "Q4納品完了" |

### 次元2: 検証（どの程度検証されているか？）

| レベル | 定義 | 信頼性 | 単独使用 |
|--------|------|--------|---------|
| 一次確認済 | 一次情報源（政府、SEC提出書類）で確認 | 最高 | 可 |
| 複数情報源 | 2つ以上の独立情報源で確認 | 高 | 可 |
| 単一信頼性 | 単一だが信頼できる情報源 | 中 | 条件付き |
| 単一未検証 | 単一情報源、未検証 | 低 | 不可 |
| 矛盾 | 複数情報源で矛盾する情報 | 注意 | 不可 |

### 次元3: 非対称性（誰が知っていて、誰が知らないか？）

| スコア | 状態 | 説明 |
|--------|------|------|
| 17-20 | 極めて高い | ほぼ未知、機関投資家も認識していない |
| 13-16 | 高い | 専門家のみ認識 |
| 9-12 | 中程度 | アナリストは認識しているが織り込み済みでない |
| 5-8 | 低い | 一部の機関投資家が認識 |
| 0-4 | なし | 広く知られ、完全に織り込み済み |

### 次元4: 鮮度（いつの情報か、どのくらい有効か？）

| 情報タイプ | 変化頻度 | 有効期間 | 更新タイミング |
|-----------|---------|----------|---------------|
| 株価、スプレッド | 日次 | 当日 | 毎回収集 |
| ポジショニング（13F） | 四半期 | 1四半期 | 四半期ごと |
| 政策発表 | 不定期 | 実行まで | イベント発生時 |
| 決算、ガイダンス | 四半期 | 次四半期まで | 決算後 |
| 市場シェア | 年次 | 6ヶ月〜1年 | 四半期確認 |
| 認証、モート | 複数年 | 複数年 | 年次確認 |

### 情報源の優先順位

#### 一次情報源（単独使用可）

| 情報源 | 例 | 優先度 |
|--------|-----|--------|
| 政府文書 | 閣議決定、官報、経産省/防衛省/文科省公式 | A |
| 国会 | 予算案、法案、国会答弁 | A |
| 有価証券報告書 | 有報、決算短信、適時開示（TDnet） | A |
| 契約・入札 | e-Gov、防衛装備庁契約情報、官公需情報 | A |

#### 二次情報源（複数で検証）

| 情報源 | 例 | 優先度 |
|--------|-----|--------|
| 業界メディア | 日経ビジネス、電子デバイス産業新聞、日刊工業新聞 | B+ |
| 通信社 | Reuters、Bloomberg、日経新聞、NHK | B |
| シンクタンク | 経済産業研究所（RIETI）、野村総研、日本総研 | B |

#### 単独使用禁止

- ソーシャルメディア（大統領含む）
- フォーラム、Reddit、個人ブログ
- アナリストの目標株価
- 未確認リーク

## 4象限マトリックス

### 軸の定義

**X軸: 確実性（高 → 右）**
- 高: 契約締結済、受注残確認済、規制承認済、長期PPA
- 低: 技術未実証、規制承認待ち、契約不確実、開発段階

**Y軸: アップサイド（高 → 上）**
- 高: 情報非対称性スコア13+、大きな市場認識ギャップ
- 低: 情報非対称性スコア8-、すでに市場に認識済み

## パフォーマンス検証サイクル

### 目的

**フレームワーク自体の妥当性を継続的に検証し、改善する。**

予測なき分析に価値はない。分析の価値は、予測の的中率で測定される。

### 検証対象と期間

| 期間 | 検証対象 | 成功基準 |
|------|---------|---------|
| **1ヶ月後** | カタリスト実現、短期株価変動 | 予測イベントの50%以上が発生 |
| **3ヶ月後** | 非対称性解消、認識トリガー | 高非対称性銘柄の20%以上で解消 |
| **6ヶ月後** | 確実性評価の的中率 | 高確実性銘柄の70%以上で想定通り |
| **1年後** | ドメインテーゼの妥当性 | Top 30 がベンチマークをアウトパフォーム |

### 評価軸の重み付け検証

| 検証項目 | 方法 | 調整トリガー |
|---------|------|-------------|
| 軸別の予測力 | 各軸スコアとリターンの相関分析 | 相関 < 0.3 の軸は重み再検討 |
| 軸間のバランス | 高スコア vs 低スコアのパフォーマンス比較 | 差異なしなら軸の有効性を再検討 |
| 閾値の妥当性 | スコア帯別のリターン分布 | 期待と乖離があれば閾値調整 |
