## 評価の順序

**「何を買うか」の前に「今買うべきか」を決める。**

```
┌─────────────────────────────────────────────────────────────┐
│ Step 1: 市場リスク分析                                      │
│         → 今は投資すべきタイミングか？                      │
│         → 攻め/守りのスタンスを決定                         │
├─────────────────────────────────────────────────────────────┤
│ Step 2: 企業評価                                            │
│         → どの企業が最も優位か？                            │
│         → 5軸スコアリング                                   │
├─────────────────────────────────────────────────────────────┤
│ Step 3: 4象限配置                                           │
│         → 確実性 × アップサイドでポジション決定             │
├─────────────────────────────────────────────────────────────┤
│ Step 4: シナリオ分析                                        │
│         → ベース/ブル/ベアでストレステスト                  │
├─────────────────────────────────────────────────────────────┤
│ Step 5: パフォーマンス検証（事後）                          │
│         → 予測は当たったか？フレームワーク改善              │
└─────────────────────────────────────────────────────────────┘
```

## 市場リスク分析

### ティア加重リスクカテゴリー（合計100点）

**根拠**: 全てのリスクが同等ではない。歴史的証拠は、日銀の政策転換と円相場変動が日本市場に最も大きな影響を与える。

#### ティア1: 最高インパクト（35点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 1 | **金融政策リスク** | 20 | 日銀政策転換（YCC修正、利上げ）、財政出動 | 2022年YCC修正、2024年マイナス金利解除 |
| 2 | **バリュエーション過熱** | 15 | 日経平均PER、TOPIX PBR、NT倍率 | 1989年バブル、2024年史上最高値 |

#### ティア2: 高インパクト（34点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 3 | **為替リスク** | 12 | ドル円水準、円相場ボラティリティ、日米金利差 | 2022年円安（150円突破）、介入リスク |
| 4 | **海外投資家動向** | 12 | 海外投資家売買動向、先物ポジション | 海外勢の売り越し/買い越しトレンド |
| 5 | **センチメント過熱** | 10 | 日経VI、騰落レシオ、信用評価損率 | 逆張り指標 |

#### ティア3: 中程度インパクト（31点）

| # | カテゴリー | 配点 | 主要指標 | 歴史的前例 |
|---|-----------|------|---------|-----------|
| 6 | **外部ショック** | 11 | 地政学（中国・台湾）、サプライチェーン | 米中摩擦、半導体規制、台湾有事リスク |
| 7 | **流動性リスク** | 10 | JGB市場、日銀保有比率、社債市場 | 日銀YCC限界、JGB流動性枯渇 |
| 8 | **業績乖離** | 10 | 株価とEPS成長のギャップ、為替前提乖離 | 想定為替レートとの乖離 |

### 歴史的クラッシュパターン（毎回確認）

| パターン | 主因 | 先行指標 | ティア |
|---------|------|---------|-------|
| 日銀政策転換 | 金融緩和終了/利上げ | 日銀会合シグナル、植田総裁発言 | **ティア1** |
| バリュエーション・クラッシュ | PER/PBRの極端な超過 | 日経平均PER > 18、PBR > 1.5 | **ティア1** |
| 円高ショック | 急激な円高進行 | 日米金利差縮小、介入観測 | ティア2 |
| 海外勢売りアンワインド | 海外投資家の大量売却 | 先物売り越し、現物売り越し継続 | ティア2 |
| 流動性危機 | JGB市場機能不全 | 日銀指値オペ、社債スプレッド拡大 | ティア3 |
| 外部ショック | 地政学リスク、災害 | 台湾有事、震災、パンデミック | ティア3 |

### 日本市場の歴史的暴落と教訓

#### 主要暴落の記録

| 暴落 | 期間 | 高値 | 安値 | 下落率 | 回復期間 | 主因 |
|------|------|------|------|--------|---------|------|
| **バブル崩壊** | 1989-2009 | 38,915円 | 7,054円 | -81.9% | **20年超** | 資産バブル崩壊、金融引き締め、不良債権 |
| **ITバブル崩壊** | 2000-2003 | 20,833円 | 7,607円 | -63.5% | 4年 | IT株過熱、米国連動 |
| **リーマンショック** | 2007-2009 | 18,261円 | 6,994円 | -61.7% | 6年 | 世界金融危機、円高 |
| **コロナショック** | 2020 | 24,083円 | 16,358円 | -32.1% | 5ヶ月 | パンデミック、一時的 |
| **2024年8月暴落** | 2024 | 42,426円 | 31,458円 | -25.8% | 3週間 | 日銀利上げ、円高、米景気不安 |

#### 暴落の教訓

```
┌─────────────────────────────────────────────────────────────────────┐
│                    日本市場暴落から学ぶ教訓                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【教訓1】金融システム毀損を伴う暴落は回復に10年以上               │
│  - バブル崩壊: 不良債権問題で「失われた20年」                       │
│  - リーマン: 銀行Tier1資本が健全なら6年で回復                       │
│  - 2024年8月: 金融システム健全なため3週間で回復                      │
│  → 銀行セクターの健全性を常に確認                                   │
│                                                                     │
│  【教訓2】日銀政策転換は最大のリスク要因                            │
│  - 1989年: 急激な金融引き締め、不動産融資総量規制                   │
│  - 2024年8月: マイナス金利解除・利上げが暴落のトリガー             │
│  → 日銀会合前後のポジション管理を徹底                               │
│                                                                     │
│  【教訓3】財務健全な企業は暴落後に強い回復                          │
│  - 無借金・高FCF企業は暴落時も配当維持・自社株買い継続             │
│  - 高レバレッジ企業は資金繰り悪化で淘汰                             │
│  → 利上げ局面ではFCF・財務健全性を重視                              │
│                                                                     │
│  【教訓4】外部ショックは一時的、構造問題は長期化                    │
│  - コロナ・2024年8月: 急落→急回復（V字）                            │
│  - バブル崩壊: 構造問題で20年超の低迷（L字）                        │
│  → ショックの性質（一時的/構造的）を見極める                        │
│                                                                     │
│  【教訓5】PER/PBRの極端な水準は警戒シグナル                         │
│  - 1989年バブル: PER 60倍超、PBR 5倍超                              │
│  - リーマン前: PER 10倍程度まで低下（割安すぎも警戒）               │
│  - 通常の下限: PER 15倍程度                                         │
│  → バリュエーション指標の極端な水準を監視                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

#### 暴落時の行動指針

| 暴落のタイプ | 対応 | 例 |
|------------|------|-----|
| 一時的ショック（金融システム健全） | **買い増し**、優良株の押し目 | 2024年8月、コロナ |
| 金融システム毀損 | **キャッシュ確保**、底打ち確認まで待機 | リーマン、バブル崩壊 |
| 日銀政策転換 | **ポジション縮小**、政策影響を見極め | 2024年8月初期 |
| 地政学ショック | **セクター精査**、影響企業を個別判断 | 台湾有事シナリオ |

### リスクスコア解釈と推奨スタンス

| スコア | 解釈 | 推奨スタンス | ターゲット企業 |
|--------|------|-------------|---------------|
| 0-25 | 低リスク | アグレッシブ | 高非対称性 + 高シナジーの日本企業 |
| 26-50 | 中リスク | バランス | 確実性とアップサイドの両立 |
| 51-75 | 高リスク | 保守的 | 高構造的優位性 + 高本質的価値 |
| 76-100 | 極端リスク | ディフェンシブ | 最高確信銘柄のみ、現金比率増 |

## 企業評価: 5軸フレームワーク（Graham分類体系統合版）

### 相対評価の原則（重要）

```
┌─────────────────────────────────────────────────────────────────────┐
│                5軸評価は相対的かつ将来予測的である                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  5軸を投資的要因(B)と投機的要因(A)に分類する:                        │
│                                                                     │
│  B. 投資的要因 (40点) — 現在確認可能な客観的事実                     │
│     軸1: 構造的優位性 (20点)                                         │
│     軸2: 本質的価値 (20点)                                           │
│                                                                     │
│  A. 投機的要因 (60点) — 将来の変化・市場認識の変動                   │
│     軸3: 将来の価値 (25点)                                           │
│     軸4: 情報非対称性 (20点)                                         │
│     軸5: 政策・触媒 (15点)                                           │
│                                                                     │
│  【評価手順】                                                        │
│  Step 1: ベーススコアを算出（絶対的な位置）                          │
│  Step 2: 相対調整（競合動向、情勢変化、前回比）                      │
│  Step 3: 情報確度(A-E)適用                                           │
│  Step 4: 最終スコアを決定                                            │
│                                                                     │
│  例: 東京エレクトロン（8035.T）                                      │
│  - 構造的優位性: 16/20（coater世界シェア90%、半導体装置参入障壁高）  │
│  - 本質的価値: 15/20（PER 25倍はセクター上限、ROE 25%+優秀）        │
│  - 投資的要因合計: 31/40                                             │
│  - 将来の価値: 20/25（AI需要 + 半導体投資の構造的増加）              │
│  - 情報非対称性: 10/20（外国人投資家に十分認知済み）                 │
│  - 政策・触媒: 12/15（半導体戦略2兆円+、TSMC熊本効果）              │
│  - 投機的要因合計: 42/60                                             │
│  - 合計: 73/100                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 5軸概要（Graham分類体系統合版・100点満点）

| 分類 | 軸 | 配点 | 核心的問い |
|------|-----|------|-----------|
| **B. 投資的要因** | 構造的優位性 | 20 | この企業なしでXはできるか？ |
| **B. 投資的要因** | 本質的価値 | 20 | 現在の株価は本質的価値に対して適正か？ |
| **A. 投機的要因** | 将来の価値 | 25 | 何ドメインの成長がこの需要を押し上げるか？ |
| **A. 投機的要因** | 情報非対称性 | 20 | この優位性は市場に認識されているか？ |
| **A. 投機的要因** | 政策・触媒 | 15 | 政策や外部イベントが価値実現を加速するか？ |

**投資的要因 (B) = 軸1 + 軸2 = 最大40点**
**投機的要因 (A) = 軸3 + 軸4 + 軸5 = 最大60点**

---

### 軸1: 構造的優位性（20点）[投資的要因]

| 要素 | 配点 | 基準 |
|------|------|------|
| **市場支配力** | 10 | 市場シェア、代替不可能性 |
| **参入障壁** | 5 | 認証、特許、習熟期間 |
| **モート財務裏付け** | 5 | R&D投資、設備投資規模 |

#### 市場支配力（10点）

| スコア | 基準 |
|--------|------|
| 9-10 | 50%以上のシェア or 重要領域唯一の供給者 |
| 7-8 | 30-50%シェア、トップ2 |
| 5-6 | 15-30%シェア、トップ3 |
| 3-4 | 10-15%シェア、差別化あり |
| 0-2 | 10%未満 or 競争優位不明確 |

#### 参入障壁（5点）

| スコア | 基準 | 日本例 |
|--------|------|--------|
| 5 | 認証10年+、技術蓄積不可逆 | 原子力規制委員会認可、防衛省認定 |
| 3-4 | 認証5-10年、高資本要件 | JAXA認定、GNT認定 |
| 1-2 | 認証3-5年、中程度 | JIS規格、業界標準 |
| 0 | 参入障壁低い | — |

#### モート財務裏付け（5点）

| スコア | 基準 |
|--------|------|
| 5 | R&D/売上10%+、設備投資で差が拡大 |
| 3-4 | R&D/売上5-10%、継続投資 |
| 1-2 | R&D投資は競合並み |
| 0 | R&D投資不足、モート縮小 |

#### 相対調整

| 条件 | 調整 |
|------|------|
| 競合がシェア拡大中 | -1〜-3 |
| 競合が撤退/弱体化 | +1〜+2 |
| 新規参入者の脅威（中国企業等） | -1〜-3 |
| 参入障壁が強化（新規制等） | +1〜+2 |

### 軸2: 本質的価値（20点）[投資的要因]

| 要素 | 配点 | 基準 |
|------|------|------|
| **収益力** | 6 | ROE、営業利益率、EPS成長 |
| **資産・資本構成** | 6 | FCF、自己資本比率、D/E比率 |
| **バリュエーション適正度** | 8 | PER、PBR、PEGとセクター比較 |

#### 収益力（6点）

| スコア | 基準 | 日本基準 |
|--------|------|---------|
| 5-6 | ROE 15%+、営業利益率20%+、EPS安定成長 | TSE Prime上位水準 |
| 3-4 | ROE 10-15%、営業利益率10-20% | TSE Prime平均以上 |
| 1-2 | ROE 5-10%、利益率改善中 | 改善トレンド確認 |
| 0 | ROE 5%未満、利益率低迷 | — |

#### 資産・資本構成（6点）

| スコア | 基準 | 日本基準 |
|--------|------|---------|
| 5-6 | FCF黒字、実質無借金、自己資本比率60%+ | 利上げ局面でも安全 |
| 3-4 | FCF黒字、D/E<1.0、自己資本比率50%+ | 標準的に健全 |
| 1-2 | FCF改善中、D/E 1.0-2.0 | 改善途上 |
| 0 | FCF赤字、高レバレッジ | 利上げリスク高 |

#### バリュエーション適正度（8点）

| スコア | 基準 | 日本基準 |
|--------|------|---------|
| 7-8 | PER<セクター中央値×0.8 & PEG<1.0 | 明確な割安 |
| 5-6 | PER≒セクター中央値 & 成長率で正当化 | 適正〜やや割安 |
| 3-4 | PER>セクター中央値だが成長プレミアム妥当 | 適正 |
| 1-2 | PER>セクター中央値×1.5、プレミアム過大 | やや割高 |
| 0 | PER>セクター中央値×2.0 | 割高（投機的） |

**日本市場の基準値（参考）:**
- 日経平均PER中央値: 約17倍（過去10年）
- TOPIX PBR中央値: 約1.2-1.4倍
- TSE Prime ROE中央値: 約9%

> **利上げ局面では軸2の資産・資本構成を特に重視。** FCF赤字かつ高レバレッジ企業は減点を強化する。

### 軸3: 将来の価値（25点）[投機的要因]

| 要素 | 配点 | 基準 |
|------|------|------|
| **シナジー** | 12 | 何ドメインの成長が需要を押し上げるか |
| **成長確度** | 8 | 投資パイプライン、TAM成長率 |
| **経営能力** | 5 | ガバナンス、資本配分、ROE改善コミットメント |

#### シナジー（12点）

各ドメインについて「投資増加でこの技術の需要は増加するか？」と問う:
- AI投資増加 → +2点
- エネルギー投資増加 → +2点
- 防衛投資増加 → +2点
- ロボティクス投資増加 → +2点
- 宇宙投資増加 → +2点
- サイバー投資増加 → +2点

最大12点（全6ドメインで需要増加）

#### 成長確度（8点）

| スコア | 基準 |
|--------|------|
| 7-8 | 5年以上の確定パイプライン（GX 150兆円、防衛43兆円） |
| 5-6 | 3-5年のパイプライン、TAM CAGR 15%+ |
| 3-4 | 明確な成長だが循環的要素あり |
| 1-2 | 成長期待はあるが不確実 |
| 0 | 成長鈍化または縮小リスク |

#### 経営能力（5点）— ガバナンス定量化フレームワーク

**参照**: Buffett（経営の質はモートと同等に重要）、Lynch（経営者の意図を理解する）、Dalio（体系的ガバナンス評価）

**概要**: 5点の配点を4つのサブコンポーネントの平均で算出。`stable/permanent/governance.json`をデータソースとする。

| サブコンポーネント | 配点 | 評価対象 |
|-------------------|------|---------|
| TSEプライム適合 | 5点満点 | PBR>1、ROE目標、独立取締役比率、英文開示 |
| 資本効率コミットメント | 5点満点 | 中計ROE目標、自社株買い履歴、配当方針、持合い解消 |
| 株主還元姿勢 | 5点満点 | 総還元性向3年平均、IR品質、アナリストカバレッジ |
| Buffett経営品質 | 5点満点 | 資本配分の実績、インサイダー保有率、CEO在任期間、ガイダンス達成率 |

**最終スコア = 4コンポーネント平均 → 0-5点（四捨五入）**

##### TSEプライム適合（5点満点）

| スコア | 基準 |
|--------|------|
| 5 | PBR>1.5、ROE>12%達成、独立取締役>1/3、英文開示あり、女性取締役あり |
| 4 | PBR>1.0、ROE目標設定・達成中、独立取締役>1/3 |
| 3 | PBR>1.0、ROE目標設定あり、改善進行中 |
| 2 | PBR<1.0だがROE改善計画あり |

#### 触媒確度の定量化（Simons式期待値計算）

**参照**: Simons（定量的確率論）、Dalio（期待値 = 確率 × マグニチュード）、Buffett（安全余裕を確率で表現）

各触媒に確率と期待値を割り当て、投資判断の精度を向上させる:

| 要素 | 定義 | 値域 |
|------|------|------|
| **probability** | 6ヶ月以内の実現確率 | 0.00〜1.00 |
| **magnitude** | スコアへの影響度 | low / medium / high |
| **expected_score_impact** | 実現時のスコア変動幅 | "+1 to +3" 等 |
| **simons_ev** | 期待値 = probability × score_impact中央値 | 数値 |

**確率ガイドライン**:

| 確率帯 | 基準 | 例 |
|--------|------|-----|
| 0.80-1.00 | 法制化済み、予算成立、契約締結 | 防衛大綱に明記された調達 |
| 0.60-0.79 | 公式発表あり、プロセス進行中 | MOU締結済みの共同開発 |
| 0.40-0.59 | 報道・検討段階、条件付き | 予算要求中、議会審議中 |
| 0.20-0.39 | 観測・推測段階 | 業界報道のみ、政策方針レベル |
| 0.00-0.19 | 不確実、長期的展望 | 構想段階、技術未実証 |

**触媒期待値（Catalyst EV）の活用**:

```
企業の触媒期待値 = Σ(各触媒のsimons_ev)

用途:
1. 同スコア企業のタイブレーカー（EV高い方を上位に）
2. 軸5スコアリングの客観的根拠
3. バックテスト時のキャリブレーション対象
4. ポートフォリオ構築時の確信度指標
```

**必須フィールド**: Top 30全社の`catalysts_next_6m`を確率付きオブジェクト配列として記録。各触媒に`probability`, `probability_basis`, `simons_ev`, `status`を含める。

### 情報確度フレームワーク（A-E）

各軸のスコアに確度レベルを付与し、低確度情報に基づくスコアを自動調整:

| レベル | 定義 | スコア乗数 | 日本の情報源例 |
|--------|------|-----------|--------------|
| A | 一次情報源で確認済 | ×1.00 | TDnet適時開示、有報、EDINET |
| B | 複数信頼できる二次情報源で一致 | ×0.90 | 日経+会社四季報で一致 |
| C | 単一の信頼できる情報源 | ×0.75 | 日経ビジネス単独報道 |
| D | 推定・計算値（クロスチェック未完了） | ×0.60 | アナリスト推定値 |
| E | 未確認・単一の非公式情報源 | ×0.40 | 業界関係者の未確認情報 |

**適用例:**
```
三菱重工（7011.T）構造的優位性
- 防衛市場シェア: 確度A（TDnet+防衛省公表）→ 10/10 × 1.00 = 10

- R&D投資: 確度B（有報+IR資料）→ 4/5 × 0.90 = 3.6 → 4（四捨五入）
- 構造的優位性合計: 19/20
```

## 情報鮮度調整（補助メカニズム）

### 5軸相対評価との関係

```
┌─────────────────────────────────────────────────────────────────────┐
│                    スコアリングの2層構造                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【主】5軸相対評価（100点満点）                                      │
│  - 各軸でベーススコア + 相対調整（競合・情勢・前回比）               │
│  - 競合の追い上げ、政策変化、ドメイン成長率の変化を反映             │
│  → これが評価の本体。相対的な変化が常に反映される。                 │
│                                                                     │
│  【補助】情報鮮度調整（-10〜+15点）                                  │
│  - 新規発見、情報陳腐化など「情報管理」の観点からの調整              │
│  → stable/蓄積量による有利/不利を調整する補助メカニズム             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 情報鮮度調整項目

```
┌─────────────────────────────────────────────────────────────────────┐
│  情報鮮度調整（-10〜+15点）                                          │
│  + 新規発見ボーナス（+5〜+10）: 市場未認識の企業                     │
│  + 不確実性減少（+2〜+5）: 噂→確定に昇格                            │
│  - 情報陳腐化（-5〜-10）: stable/の長期未更新                        │
│  - 変化なし長期化（-3〜-5）: 3回以上変動なし                         │
└─────────────────────────────────────────────────────────────────────┘
```

### 動的調整項目

#### 1. 新規発見ボーナス（+5〜+10点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 今回初めて発見した企業 | +10 | 市場がまだ認識していない可能性大 |
| 過去3回以内に発見した企業 | +5 | まだ認知度が低い |
| stable/に6ヶ月以上蓄積 | 0 | すでに評価が安定 |

#### 2. 重要変化検知（+3〜+5点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 変化アラート発生（市場シェア、モート） | +5 | 市場が追いついていない変化 |
| 新規大型契約（$100M+） | +5 | カタリスト |
| 経営層変更、M&A | +3 | 潜在的転機 |

#### 3. 不確実性減少（+2〜+5点）

| 条件 | ボーナス | 理由 |
|------|---------|------|
| 噂→公式発表に昇格 | +5 | リスク大幅減 |
| 未確認→確認に昇格 | +3 | 信頼度向上 |
| 一次情報源で検証完了 | +2 | 確度確定 |

#### 4. 情報陳腐化（-5〜-10点）

| 条件 | ペナルティ | 理由 |
|------|-----------|------|
| stable/の最終更新が6ヶ月以上前 | -5 | 情報の鮮度懸念 |
| stable/の最終更新が12ヶ月以上前 | -10 | 大幅な陳腐化リスク |
| 今回LLM収集で言及なし | -3 | 関心低下の可能性 |

#### 5. 変化なし長期化（-3〜-5点）

| 条件 | ペナルティ | 理由 |
|------|-----------|------|
| 3回連続でスコア変動なし | -3 | 動きがない=機会が少ない |
| 6回連続でスコア変動なし | -5 | 硬直化、見直し必要 |

### 適用例

```
例1: 新規発見の小型企業
- 5軸ベーススコア: 65点（情報不足で控えめ評価）
- 新規発見ボーナス: +10
- 重要変化検知: +5（大型契約発表）
→ 最終スコア: 80点

例2: stable/に長期蓄積の大型企業
- 5軸ベーススコア: 85点（詳細情報で高評価）
- 情報陳腐化: -5（8ヶ月更新なし）
- 変化なし長期化: -3
→ 最終スコア: 77点

→ 新規発見企業が大型企業を逆転する可能性あり
```

|:---------------:|---------|:------------:|:--------------------|
| 0-25 | 積極的 | 5-10% | 全体100%、投機象限100% |
| 26-50 | バランス | 10-15% | 全体100%、投機象限80% |
| 51-75 | 保守的 | 15-25% | 全体80%、投機象限50% |
| 76-100 | 防御的 | 25-40% | 全体50%、投機象限0% |

### 複合エグジットスコア

5つの要因を加重して0.0〜1.0の複合スコアを計算:

| 要因 | 重み | トリガー条件 |
|------|:----:|-------------|
| ランク低下 | 0.30 | 5ランク以上低下 |
| 触媒失敗 | 0.20 | catalysts_next_6mの実現率が40%未満 |
| スコア連続低下 | 0.20 | 3回連続でスコア低下 |
| 高市場リスク | 0.15 | 市場リスクスコア > 60 |
| テーゼ無効化 | 0.15 | 核心技術の代替出現、主要契約喪失、認証取消 |

**複合スコア解釈**:
- 0.00-0.30: 保有継続
- 0.31-0.50: 監視強化
- 0.51-0.70: ポジション縮小検討
- 0.71-1.00: エグジット推奨

### Soros再帰性チェック

各分析サイクルで「投資テーゼのナラティブは健在か？」を検証:

```
チェック項目:
1. テーゼを支える主要前提は変わっていないか？
2. 反証（カウンターエビデンス）は増えていないか？
3. 市場のナラティブが自己強化ループに入っていないか？
   → 過度な楽観: バブル懸念
   → 過度な悲観: 逆張り機会
```

### テーゼ無効化基準

以下のいずれかに該当する場合、テーゼ無効化と判定:

| 基準 | 例 |
|------|-----|
| 核心技術が代替される | レーザーテックの検査技術に競合出現 |
| 主要契約の喪失（代替なし） | 防衛プライム契約の喪失 |
| 認証の取消 | 防衛・原子力認証の失効 |
| 競合がシェアの50%以上を獲得 | 新興国企業による急速なシェア侵食 |
| モートの構造的崩壊 | 技術標準の変更によるロックイン解除 |

## ポートフォリオ構築フレームワーク

**参照**: Dalio（リスクパリティ、非相関資産への分散）、Buffett（高確信度への集中）、Lynch（GARP型ポジションサイジング）、Simons（体系的配分）

### 配分の原則

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ポートフォリオ構築の3原則                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  原則1: 確信度に比例したサイジング（Buffett）                       │
│  - Tier 1（最高確信度）= 最大ウェイト                               │
│  - Tier 4（投機的）= 最小ウェイト                                   │
│                                                                     │
│  原則2: リスク分散（Dalio）                                         │
│  - ドメイン集中上限: 30%                                            │
│  - 単一銘柄上限: 8%                                                 │
│  - 相関の高い銘柄群の合計上限: 25%                                  │
│                                                                     │
│  原則3: 市場リスク連動（全天候型）                                   │
│  - 市場リスク高→キャッシュ増、ポジション縮小                       │
│  - 市場リスク低→キャッシュ減、ポジション拡大                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### ポジションサイジング公式

```
weight = base_weight × tier_multiplier × quadrant_multiplier × market_risk_factor × exit_signal_adj

パラメータ:
  base_weight = 1 / 持株数（均等配分ベース）

  tier_multiplier:
    Tier 1 = 1.5  （Top 10、最高確信度）
    Tier 2 = 1.0  （Top 11-20、高確信度）
    Tier 3 = 0.7  （Top 21-30、中確信度）
    Tier 4 = 0.3  （Top 30外の監視銘柄）

  quadrant_multiplier:
    最優先(prime) = 1.2
    安定成長(stable) = 1.0
    投機的(speculative) = 0.5

  market_risk_factor:
    0-25 → 1.2  （積極的）
    26-50 → 1.0  （バランス）
    51-75 → 0.8  （保守的）
    76-100 → 0.5  （防御的）

  exit_signal_adj:
    none = 1.0
    yellow = 0.5
    red = 0.0

正規化: 全ウェイトの合計が(1-キャッシュ比率)になるよう調整
```

### 象限別配分ターゲット

| 象限 | 配分ターゲット | レンジ | 1銘柄上限 |
|------|:------------:|:------:|:--------:|
| 最優先 | 45% | 35-55% | 8% |
| 安定成長 | 25% | 20-35% | 6% |
| 投機的 | 10% | 5-15% | 3% |
| キャッシュ | 20% | 15-25% | — |

### 制約条件

| パラメータ | 上限 | 根拠 |
|-----------|:----:|------|
| 単一銘柄 | 8% | 集中リスク回避（Buffett: 高確信は集中、ただし上限あり） |
| ドメイン集中 | 30% | セクターリスク分散（Dalio: 非相関への分散） |
| 投機象限合計 | 15% | テール・リスク管理 |
| 相関銘柄群 | 25% | 同一供給チェーン内の集中回避 |
| 保有銘柄数 | 15-25 | 分散と管理可能性のバランス（Lynch） |

### 入力と出力

- **入力**: `data/portfolio/holdings.json`（ユーザー提供の現保有データ）
- **出力**: `data/portfolio/allocation_model.json`（推奨配分）、`data/portfolio/rebalance_recommendations.json`（リバランスアクション）

> **注**: 推奨配分はAIが生成するが、最終的な売買判断は人間が行う（意思決定マトリックス参照）。

### 金利上昇環境での優先事項


## パフォーマンス検証サイクル（バックテスト統合版）

### 自動バックテストプロトコル

検証スケジュール日を経過した場合、`verify performance [1m|3m|6m|1y]`で自動実行:

```
バックテスト実行手順:
1. predictions/[DATE]/から予測スナップショットを読み込み
2. Web検索で実績データを収集（株価、触媒結果、イベント）
3. actuals/[DATE]/に実績データを記録
4. 予測 vs 実績を比較分析
5. backtest_results.jsonを生成
6. axis_effectiveness.jsonで軸予測力を分析
7. framework_calibration.jsonで調整推奨を生成
```

#### 軸有効性分析（Simons式）

各軸のスコアと実際のリターンの相関を分析:

| 分析項目 | 方法 | 意味 |
|---------|------|------|
| 相関係数 | 軸スコア vs 3ヶ月リターン | 軸の予測力 |
| スプレッド | 上位10銘柄リターン - 下位10銘柄リターン | 軸の弁別力 |
| 調整トリガー | 相関 < 0.3 | 重み再検討が必要 |

#### 触媒キャリブレーション

予測確率と実際の実現率を比較し、確率推定の精度を検証:

| 予測確率帯 | 期待実現率 | キャリブレーション |
|-----------|:--------:|:---------------:|
| 70%+ | 65-80% | 適正 |
| 40-69% | 35-65% | 適正 |
| < 40% | 20-45% | 適正 |

実現率が期待から大幅に乖離 → 確率推定方法論を改善

#### エグジットシグナル検証

| 検証項目 | 指標 | 目標 |
|---------|------|:----:|
| レッドアラートの正当性 | 発行後のリターン < 0% | > 70% |
| イエローアラートの正当性 | 発行後のリターン < ベンチマーク | > 50% |
| 偽陽性率 | アラート発行したが回復した比率 | < 40% |
| 見逃し率 | アラートなしで大幅下落した比率 | < 20% |

### システムリスクの監視

フレームワーク自体のリスクも監視する。

| リスク | 監視指標 | 対処 |
|--------|---------|------|
| LLM出力品質劣化 | 矛盾率の推移、情報の再現性 | 品質低下時はLLM変更を検討 |
| 情報源の信頼性低下 | 一次情報源での検証失敗率 | 情報源リストの見直し |
| フレームワークのバイアス | 特定ドメイン/企業への偏り | 定期的なバイアス監査 |
| 過剰適合 | 過去データへの依存度 | Out-of-sampleテストの実施 |
